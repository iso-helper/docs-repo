ISO 21001:2018 管理評審報告大綱：

1.簡介：

評審日期：指定管理評審發生的日期。
出席者：列出所有出席者，包括他們的角色和職責。
目的：說明管理評審的目的（例如，評估品質管理系統在實現預期結果方面的有效性）。

2. 品質管理系統（QMS）審查：

績效：根據計畫目標分析 QMS 的績效。包括與以下內容相關的關鍵績效指標 (KPI) 和指標：
客戶滿意度
符合要求
流程的有效性
持續改進
資源分配
內部稽核的有效性
糾正和預防措施的有效性
管理評審過程本身的有效性
內部稽核結果：總結最近內部稽核的結果以及採取的任何糾正措施。
不合格情況和糾正措施：審查發現的任何不合格情況以及實施的糾正措施的有效性。
客戶回饋：總結客戶對所提供服務品質的回饋。
背景變化：解決組織背景（內部或外部）中可能影響 QMS 的任何變化。
風險和機會：討論與 QMS 相關的已識別風險和機遇，以及已實施的緩解和利用計劃的有效性。

3. 目標審查：

目標設定檢討：評估先前的管理審查中設定的目標的實現。
設定新目標：根據審查結果提出 QMS 的新目標。這些應該是 SMART（具體、可衡量、可實現、相關、有時限）。

4. 建議和行動：

所需行動：確定解決任何缺陷或改進 QMS 所需的行動。
責任分配：為每個行動項目分配責任。
目標完成日期：為每個行動項目設定目標完成日期。

5.結論：

總結管理評審的主要發現。
說明 QMS 的整體有效性。
確認致力於持續改善 QMS。

6.附錄（如有必要）：

支援文檔，例如資料圖表、圖形或 KPI 的詳細分析。
