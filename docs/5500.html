本程序文件規範倉儲及運輸管理之流程，以確保貨物安全、有效率地儲存及運送。

一、適用範圍

本程序適用於所有公司內部之倉儲及運輸作業，包含但不限於：收貨、驗收、入庫、儲存、出庫、裝卸、運輸及交付等環節。

二、相關責任

  • 倉儲管理員: 負責倉儲作業之日常管理，包含貨物之收發、驗收、入庫、儲存、出庫及庫存管理等。
  • 運輸部門: 負責貨物之運輸安排、車輛調度及送貨等。
  • 相關部門: 例如採購、銷售等部門需配合提供相關資訊，以確保儲運作業順利進行。

三、作業流程

(一) 收貨及驗收

1. 收貨人員根據訂單或通知單，核對貨物數量、品名、規格及包裝等資訊。
2. 驗收貨物，檢查是否有損壞、短缺或品質問題。
3. 如有異常情況，需立即記錄並通知相關部門處理。
4. 完成驗收後，簽署收貨單據。

(二) 入庫

1. 根據貨物特性及庫存狀況，選擇合適的儲存位置。
2. 使用條碼或其他識別方式，追蹤貨物位置。
3. 正確記錄入庫資訊，包含貨物名稱、數量、批號、入庫日期及位置等。

(三) 儲存

1. 根據貨物特性，選擇適當的儲存方式及環境條件(例如溫度、濕度)。
2. 定期檢查貨物狀況，避免損壞或變質。
3. 實施先進先出(FIFO)原則，避免貨物過期。
4. 維持倉庫整潔，確保安全。

(四) 出庫

1. 根據出庫單據，核對貨物數量、品名及規格等資訊。
2. 從指定位置提取貨物。
3. 檢查貨物狀況，確保無損壞。
4. 完成出庫手續，更新庫存記錄。

(五) 裝卸

1. 使用適當的裝卸設備及工具，確保貨物安全。
2. 避免貨物損壞或人員受傷。
3. 遵守安全操作規範。

(六) 運輸

1. 選擇合適的運輸方式及承運商。
2. 確保貨物安全及準時送達。
3. 全程追蹤貨物運送狀況。
4. 妥善處理運輸過程中發生的問題。

(七) 交付

1. 核對收貨人資訊及貨物資訊。
2. 取得收貨人的簽收。
3. 完成交付手續。

四、文件管理

  • 所有相關文件，例如收貨單、入庫單、出庫單、運輸單據等，應妥善保管，並依公司規定保存期限。

五、安全管理

1. 遵守相關安全規範及操作程序。
2. 定期進行安全檢查及培訓。
3. 處理任何安全事故及意外事件。

六、持續改進

  • 本程序應定期檢討及更新，以確保其有效性及適時性。
