本程序旨在建立一套系統化的職業災害事故調查及處理機制，以有效預防類似事故再次發生，保障勞工安全與健康。

一、通報及應變 (立即處理)

1. 事故發生通報: 事故發生後，相關人員應立即通報主管及相關單位 (例如：安全衛生管理人員、公司管理階層、主管機關等)，並依據公司既定程序，儘速通報主管機關。
2. 緊急救護: 優先進行受傷人員的緊急救護，必要時撥打119或其他緊急救護電話。 確保現場安全，防止事故擴大。
3. 現場保護: 在確保安全的前提下，儘可能保持事故現場原狀，避免破壞相關證據。 拍照或錄影記錄現場狀況，並製作初步現場記錄。
4. 人員安置: 安排目擊者及相關人員休息，避免造成二次傷害或心理衝擊。 提供必要的心理輔導協助。

二、事故調查 (深入調查)

1. 成立調查小組: 由公司指派相關人員組成調查小組，成員應包括安全衛生管理人員、相關部門主管、技術人員及其他相關專業人士。 必要時，可邀請外部專家參與調查。
2. 調查範圍: 調查應涵蓋事故發生的時間、地點、經過、人員傷亡情況、事故原因、相關設備及作業流程等。
3. 證據蒐集: 蒐集相關證據，包括但不限於：事故現場照片、錄影、目擊證詞、相關文件記錄、設備維護記錄、安全檢查記錄等。
4. 原因分析: 利用因果分析圖、魚骨圖等方法，分析事故發生的根本原因，並找出潛在的危險因子。
5. 撰寫調查報告: 調查小組應根據調查結果，撰寫詳細的調查報告，包含事故經過、原因分析、責任歸屬、預防措施等。 報告應清晰、完整、客觀，並附上相關證據。

三、事故處理 (改善預防)

1. 糾正措施: 根據調查報告，提出具體的糾正措施，以消除事故發生原因及潛在危險因子。
2. 預防措施: 針對事故原因及潛在危險因子，制定預防措施，以避免類似事故再次發生。 此包括：改進作業流程、更新設備、加強安全教育訓練、改善工作環境等。
3. 執行與追蹤: 確實執行糾正及預防措施，並定期追蹤成效，確保措施的有效性。
4. 後續追蹤: 對受傷人員進行後續追蹤，提供必要的醫療及復健協助。 定期追蹤其身心健康狀況。
5. 報告提交: 將事故調查報告及處理措施提交相關單位，包括公司管理階層及主管機關。

四、定期檢討與改善

  • 公司應定期檢討職業災害事故調查及處理程序，並根據實際情況進行調整與改善，以提升職業安全衛生管理效能。 建立持續改進機制，預防職業災害事故發生。
