目的: 建立一套標準程序，有效控制及處理不合格品，以防止不合格品流入客戶手中，降低損失，並持續改進產品及製程品質。

範圍: 本程序適用於所有生產、採購、銷售等環節發現的不合格品。

定義:

  • 不合格品 (Nonconforming Product): 未能滿足規定要求的產品、材料、零組件、服務或過程。
  • 不合格原因 (Cause of Nonconformance): 導致產品或過程不符合規定要求的原因。
  • 矯正措施 (Corrective Action): 消除不合格品原因的措施。
  • 預防措施 (Preventive Action): 防止類似不合格再次發生的措施。
  • 隔離 (Segregation): 將不合格品與合格品隔離，防止混淆或誤用。
  • 處置 (Disposition): 對不合格品的最終處理方式，例如修復、報廢、讓步接收等。

程序:

1. 不合格品的識別與隔離:

  • 所有人員應對不合格品保持高度警覺，並立即採取行動。
  • 發現不合格品時，應立即停止相關作業，並使用明顯標記（例如：不合格標籤）進行隔離，防止其繼續流入後續製程。
  • 記錄不合格品的相關資訊，包括：識別號碼、數量、不合格類型、發現時間、發現地點、發現人員等。

2. 不合格品報告:

  • 將不合格品相關資訊填寫於不合格品報告表 (附件一)。
  • 將報告表提交至品質管理部門或指定人員。

3. 不合格品審查:

  • 品質管理部門或指定人員對不合格品報告進行審查，確定不合格原因。
  • 必要時，可進行現場調查及分析。

4. 不合格品處置:

根據不合格品的嚴重程度、成本效益及客戶要求，選擇合適的處置方式：

  • 修復 (Repair): 如果經濟上可行且修復後能滿足要求，則進行修復。修復後需進行驗證。
  • 返工 (Rework): 對不合格品進行重新加工。返工後需進行驗證。
  • 降級 (Downgrade): 將不合格品降級使用，但需確保符合降級後的規格要求。
  • 讓步接收 (Concession): 在特定條件下，經授權人員批准後，允許接受不合格品。
  • 報廢 (Scrap): 如果修復或其他處置方式不經濟或不可行，則將不合格品報廢。報廢需記錄並依規定處理。

5. 矯正及預防措施:

  • 針對不合格原因，制定並實施矯正措施，以消除不合格品發生的根本原因。
  • 制定並實施預防措施，以防止類似不合格再次發生。
  • 矯正和預防措施的有效性應進行驗證。

6. 記錄:

  • 所有不合格品相關記錄應妥善保存，並按照公司文件管理規定進行管理。

責任:

  • 各部門負責人負責監督本程序在部門內的執行。
  • 品質管理部門負責本程序的制定、修訂和執行。
