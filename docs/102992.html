1. 目的

本程序書旨在描述管理與生產過程及產品相關變更的系統性方法，以確保符合 ISO 22716:2007 (GMP 規範良好生產規範) 的要求，並維持產品質量和一致性。本程序適用於所有可能影響產品質量、安全性和有效性的變更。

2. 範圍

本程序適用於所有與生產過程和產品相關的變更，包括但不限於：

原料、包裝材料和輔料的變更
 配方和製程的變更
設備和儀器的變更
 生產場所和設施的變更
人員的變更 (例如：關鍵崗位人員的更換)
 質量管理系統的變更
測試方法的變更
 供應商的變更

3. 職責

變更提議者: 負責提交詳細的變更提議，包括變更的理由、影響評估和建議的補救措施。
 變更評估小組: 負責評估變更提議，確定潛在的影響，並批准或拒絕變更。此小組應包括相關部門的代表，例如生產、質量保證、研發和法規事務。
變更執行者: 負責根據批准的變更計劃執行變更，並確保變更有效執行。
 質量保證部門: 負責監控變更的執行，並驗證變更是否符合要求。

4. 程序

4.1 變更提議:

1. 變更提議者需填寫變更控制申請表 (附件1)，詳細說明變更內容、理由、預期影響（包括產品質量、安全性、有效性和生產效率）以及建議的補救措施。
2. 申請表需提交給變更評估小組進行審查。

4.2 變更評估:

1. 變更評估小組審查變更提議，評估其對產品質量、安全性、有效性和合規性的潛在影響。
2. 評估小組需進行風險評估，並確定必要的驗證活動。
3. 評估小組根據評估結果，決定批准或拒絕變更。
4. 批准的變更需編號並記錄在變更登記冊 (附件2)。

4.3 變更執行:

1. 變更執行者根據批准的變更計劃執行變更。
2. 執行過程中需嚴格遵守標準操作程序 (SOP) 和其他相關文件。
3. 執行過程需記錄在案，包括日期、時間、執行者、以及任何偏差。

4.4 變更驗證:

1. 質量保證部門負責驗證變更的有效性，並確認變更後的產品或過程符合規定的要求。
2. 驗證活動應包括對產品或過程的測試和檢查。

4.5 文件記錄:

所有變更相關文件，包括變更提議、評估報告、執行記錄和驗證報告，都應妥善保存和管理。

5. 偏差處理

任何在變更執行過程中發現的偏差，都應立即記錄並報告給變更評估小組。偏差需進行調查，並制定糾正和預防措施。

6. 審核

本程序書將定期進行審核，以確保其有效性和適宜性。
