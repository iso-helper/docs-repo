一、廢棄物分類與源頭減量:

  • 分類: 依據廢棄物種類進行分類，常見分類包含：一般垃圾、資源回收物(紙類、塑膠類、玻璃瓶罐、金屬類、寶特瓶)、廚餘、有害垃圾(例如：電池、燈管、油漆)、大型廢棄物(例如：家具、家電)。 分類越精細，回收再利用的效率越高。
  • 源頭減量: 在產生廢棄物之前，就盡力減少廢棄物的產生量。例如：減少使用一次性用品、重複使用物品、選擇環保包裝等。 這是廢棄物處理中最重要的一環，能大幅減輕後續處理的負擔。

二、廢棄物收集:

  • 收集容器: 使用適當的容器分類收集不同類型的廢棄物，並標示清楚。
  • 收集頻率: 依據廢棄物種類和產生量決定收集頻率，例如：一般垃圾每日收集，資源回收物每週收集。
  • 收集方式: 包含自行分類投遞至資源回收站、委託清潔隊收運、或企業自行處理等方式。

三、廢棄物運送:

  • 運送車輛: 使用符合規定的運送車輛，並確保廢棄物在運送過程中不會造成環境污染。
  • 路線規劃: 規劃有效的運送路線，以提高效率並降低運送成本。
  • 安全措施: 確保運送過程的安全，避免發生意外事故。

四、廢棄物處理:

  • 一般垃圾: 通常送至掩埋場進行掩埋處理，部分地區會進行焚化處理，產生能量。
  • 資源回收物: 送至資源回收廠進行分類、清洗、加工，再製成再生資源。
  • 廚餘: 可以進行堆肥、厭氧消化等處理，產生堆肥或沼氣。
  • 有害垃圾: 送至專門的處理廠進行特殊處理，例如：焚化、化學處理等，以避免造成環境污染。
  • 大型廢棄物: 通常需要專人處理，有些可以直接回收再利用，有些則需進行破碎或焚化處理。

五、廢棄物處理設施管理:

  • 環境監測: 定期監測處理設施的環境影響，確保符合相關法規。
  • 設備維護: 定期維護處理設施的設備，以確保其正常運作。
  • 人員訓練: 對處理人員進行專業訓練，以提高其處理效率和安全意識。

六、法規遵循:

  • 遵守相關的廢棄物管理法規，例如：資源回收再利用法、廢棄物清理法等。
  • 取得相關的許可證或執照。
