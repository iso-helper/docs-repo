一、目的

為確保文件之完整性、正確性及有效性，建立一套標準作業程序，規範文件之建立、發行、修訂、儲存、查閱、銷毀等流程，以提升文件管理效率及降低風險。

二、適用範圍

本標準作業程序適用於所有由(單位名稱)產生、使用及管理的文件，包含但不限於：

  • 文件類型：文件、表格、圖表、電子郵件、報告、記錄、程序文件、規範、標準等。
  • 文件格式：紙本文件、電子文件等。

三、定義

  • 文件: 指任何記錄資訊的媒介，包括紙本、電子、錄音或錄影等形式。
  • 原始文件: 首次產生之文件。
  • 修訂文件: 對原始文件進行修改後之文件。
  • 發行文件: 正式發布並可供使用的文件。
  • 檔案管理員: 負責文件管理相關工作之人員。
  • 文件版本號: 用於識別文件修訂版本的編號。

四、程序

(一) 文件建立

1. 文件創建者需使用標準化的模板（如適用）。
2. 文件內容應清晰、準確、完整，並包含必要的資訊（例如：文件編號、版本號、作者、日期、審核人等）。
3. 文件建立後，應由相關人員審核確認。
4. 審核通過的文件需經授權者批准後方可發行。

(二) 文件發行

1. 獲得授權後，文件管理員負責文件發行。
2. 發行方式：可依文件性質選擇紙本或電子方式發行。
3. 發行後，需記錄發布日期及發布對象。

(三) 文件修訂

1. 文件需修訂時，需填寫修訂記錄表，說明修訂內容及原因。
2. 修訂文件需重新進行審核及授權程序。
3. 修訂後的文件應標明新的版本號。
4. 需明確標示舊版文件作廢，並妥善保管。

(四) 文件儲存

1. 文件儲存場所應安全、整潔、易於查找。
2. 紙本文件需妥善保管，避免損壞或遺失。
3. 電子文件應儲存於指定的伺服器或雲端儲存空間，並定期備份。
4. 文件儲存應遵循公司資料安全管理規範。

(五) 文件查閱

1. 文件查閱應符合公司保密規定。
2. 查閱者需簽名或登記。
3. 禁止複製、攜出未經授權的文件。

(六) 文件銷毀

1. 文件銷毀需依公司規定執行。
2. 銷毀前需經授權者批准。
3. 銷毀方式應確保文件內容無法恢復。
4. 需記錄銷毀日期、方式及相關人員。
