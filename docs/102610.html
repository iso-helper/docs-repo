1. 目的

本程序書旨在定義採購石油、天然氣、石化及相關行業材料、設備和服務的流程，確保符合ISO 29001標準的要求，並滿足組織的質量、安全、環保和交付要求。

2. 範圍

本程序書適用於所有採購材料、設備和服務的活動，包括但不限於：

原材料 (例如：鋼材、塑膠、化學品)
 設備和儀器 (例如：泵、閥門、分析儀器)
服務 (例如：維護、檢驗、培訓)
 軟體和技術支援

3. 責任

採購部門: 負責執行本程序書中定義的所有活動。
 相關部門: 負責提供採購需求、技術規格及驗收標準。
管理層: 負責確保本程序書的有效執行和持續改進。

4. 程序

4.1 需求識別與規格制定:

相關部門根據項目需求，制定明確的採購需求說明書，包含：
材料、設備或服務的詳細描述及技術規格。
 質量要求，包括相關標準或規範。
安全要求，包括相關安全標準或規範。
 環境要求，包括相關環保標準或規範。
交付期限和地點。
 數量和單位。
驗收標準及測試方法。
 預算限制。
需求說明書需經相關部門審核批准。

4.2 供應商選擇與評估:

根據預先確定的評估標準（例如：質量管理體系認證、安全記錄、環境績效、財務狀況、能力及經驗等），評估潛在供應商。
考慮使用供應商名錄或供應商數據庫。
 進行供應商審核，必要時進行現場評估。
選擇符合要求的合格供應商。

4.3 採購訂單發放:

根據批准的採購需求說明書和選定的供應商，發放正式的採購訂單。
採購訂單需包含所有相關信息，例如：
 訂單號碼。
供應商信息。
 材料、設備或服務的詳細描述。
數量和單價。
 交付期限和地點。
付款方式。
 相關質量、安全和環保要求。
違約條款。

4.4 供應商管理:

定期監控供應商的績效，包括：交付時間、質量、安全和環保表現。
與供應商保持良好的溝通，及時解決任何問題。
 必要時，對供應商進行績效評估和改進措施。

4.5 材料、設備和服務的驗收:

根據採購訂單和驗收標準，對收到的材料、設備和服務進行驗收檢查。
 驗收檢查包括：
文件審查 (例如：質量證明書、檢驗報告)。
 物理檢查 (例如：尺寸、重量、外觀)。
功能測試 (例如：性能測試、可靠性測試)。
 完成驗收記錄。

4.6 採購記錄:

保持完整的採購記錄，包括：
 採購需求說明書。
供應商評估記錄。
 採購訂單。
驗收記錄。
 供應商績效記錄。

5. 記錄控制

所有記錄均應按照公司的記錄管理程序進行管理和保存。

6. 程序修訂

本程序書將定期審閱和修訂，以確保其持續適用性。
