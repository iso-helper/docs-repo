目的: 建立一套完善的產品回收管制程序，以確保公司符合相關法規，並有效率地處理產品回收，降低環境負擔及維護公司聲譽。

適用範圍: 本程序適用於所有公司生產、銷售及進口的產品回收作業。

程序:

一、產品回收識別與分類:

1. 回收產品識別: 產品設計階段即需考慮產品生命週期結束後的回收及再利用，並在產品上標示相關回收資訊（例如：回收標誌、材質標示等）。
2. 回收產品分類: 依據產品材質、種類、危險性等因素，將回收產品分類，以便於後續處理。
3. 回收產品登記: 建立回收產品登記系統，記錄回收產品的數量、種類、來源、回收日期及處理方式等資訊。

二、回收渠道建立與管理:

1. 回收渠道規劃: 規劃多元的產品回收渠道，例如：公司自行回收、委託第三方回收商、設立回收點等，以方便消費者進行產品回收。
2. 回收渠道宣傳: 透過公司網站、產品包裝、媒體廣告等方式，向消費者宣傳產品回收資訊及回收渠道。
3. 回收渠道合作夥伴管理: 建立與回收合作夥伴的合作協議，明確雙方的責任與義務，確保回收作業的順利進行。 定期審核合作夥伴的績效及合規性。

三、回收作業流程:

1. 回收產品接收: 建立標準化的產品回收接收程序，包括產品檢查、分類、記錄等步驟。
2. 回收產品處理: 依據產品種類及材質選擇適當的處理方式，例如：拆解、再利用、再循環、焚燒（符合環保規範）等。
3. 回收記錄管理: 詳細記錄回收產品的處理過程，包括處理日期、處理方式、處理數量及處理結果等。
4. 廢棄物管理: 確保回收過程產生的廢棄物符合相關法規，並委託具有合格證照的廢棄物處理業者進行處理。

四、績效評估與持續改善:

1. 回收率統計: 定期統計產品回收率，追蹤回收成效。
2. 程序效能評估: 定期檢討回收程序的效能，並根據實際情況進行調整與改善。
3. 法規遵循性檢視: 定期檢視相關法規及標準，確保回收程序符合最新規定。
4. 風險評估與管理: 定期進行風險評估，並制定相應的風險管理措施，以防範可能的環境污染及安全事故。

五、緊急應變程序:

  • 針對回收作業中可能發生的意外事故（例如：洩漏、火災等），制定完善的緊急應變程序，並定期進行演練。
