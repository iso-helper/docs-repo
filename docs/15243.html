一、 範圍界定:

1. 確認適用法規範圍: 明確界定需要進行法規鑑別的對象，例如：特定的工廠、部門、作業流程或特定設備。
2. 識別相關活動: 列出所有在範圍界定內進行的活動，例如：製造、搬運、儲存、維修等。
3. 確認所有員工類別: 識別所有可能接觸到危害的員工類別，例如：操作員、維修人員、管理人員等，不同員工類別可能適用不同的法規。

二、 法規資料蒐集:

1. 參考法規清單: 利用政府官方網站、專業資料庫或顧問協助，蒐集可能適用的安全衛生相關法規、標準、規範、指引等。 這包括但不限於：勞工安全衛生法及其相關子法規、其他相關產業法規、地方政府相關規定等。
2. 關鍵字搜尋: 使用與特定活動、物質或設備相關的關鍵字進行搜尋，以確保沒有遺漏任何相關法規。
3. 定期更新法規資料庫: 法規會不斷更新，因此需要定期更新資料庫，確保所使用的法規是最新的版本。

三、 法規分析與評估:

1. 比對法規與實際作業: 將蒐集到的法規與實際的作業流程、使用的設備、物質等進行比對，判斷哪些法規適用於該工作場所。
2. 風險評估: 進行風險評估，判斷各項危害的可能性和嚴重性，並依據風險程度選擇相應的防護措施和法規遵循。
3. 優先級排序: 根據風險程度及法規的強制性，將法規進行優先級排序，先處理高風險、高強制性的法規。
4. 文件記錄: 詳細記錄法規鑑別的過程、結果以及依據，以便日後追蹤和審查。

四、 法規遵循與執行:

1. 制定安全衛生管理制度: 根據鑑別結果，制定相應的安全衛生管理制度、操作規範、安全教育訓練計畫等。
2. 提供必要的防護措施: 提供符合法規要求的個人防護設備 (PPE) 、安全設備及環境改善措施。
3. 定期檢查與稽核: 定期檢查安全衛生措施的執行情況，並進行稽核以確保符合法規要求。
4. 持續改善: 根據檢查和稽核結果，持續改善安全衛生管理系統，並定期更新法規鑑別結果。

五、 持續監控與更新:

1. 法規變更追蹤: 持續關注法規的更新和變動，並及時調整安全衛生管理系統。
2. 事故分析與經驗學習: 發生事故時，需進行徹底的事故調查分析，找出事故原因，並改善安全衛生管理系統，防止類似事故再次發生。
