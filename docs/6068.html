目的: 建立一套完善的設備維護及工作環境管理程序，以確保設備正常運作、延長設備壽命、提升生產效率，並創造安全、舒適、清潔的工作環境，預防職業災害，保障員工身心健康。

範圍: 本程序適用於所有公司設備及工作場所。

責任:

  • 設備管理員: 負責設備的日常檢查、保養、維修及記錄。
  • 部門主管: 負責監督所屬部門設備的維護及工作環境的管理，並確保員工遵守相關規定。
  • 所有員工: 負責維持自身工作區域的清潔與秩序，並正確使用設備及工具，報告任何設備異常或環境問題。
  • 安全衛生管理人員: 負責監督本程序的執行，並定期進行檢查及評估。

程序:

一、設備維護:

1. 預防性維護:

  • 定期檢查：依據設備維護計劃表，定期對設備進行檢查，記錄檢查結果及異常狀況。
  • 清潔保養：定期清潔設備，並根據設備說明書進行潤滑、調整等保養工作。
  • 潤滑保養：按時進行設備潤滑，使用指定潤滑油。
  • 預防性更換：定期更換易損件，避免因零件老化造成設備故障。

2. 矯正性維護:

  • 故障排除：設備發生故障時，應立即停機並進行故障排除，必要時聯繫專業維修人員。
  • 維修記錄：詳細記錄故障原因、維修過程、維修人員及維修日期等資訊。
  • 零件管理：妥善保管維修零件，並建立零件庫存管理系統。

3. 設備維護計劃:

  • 制定年度設備維護計劃，明確維護項目、時間、人員及預算。
  • 定期檢討並更新維護計劃，以符合實際需求。

4. 設備標識:

  • 所有設備應清楚標示設備名稱、編號及相關注意事項。

二、工作環境管理:

1. 清潔與整理:

  • 定期清潔工作場所，保持地面乾燥、整潔。
  • 清理垃圾及廢棄物，並按照規定分類處理。
  • 定期消毒，預防傳染病的發生。

2. 安全管理:

  • 確保工作場所的安全設施完善，如安全通道、消防設備等。
  • 按照規定穿戴個人防護設備 (PPE)，例如安全帽、安全鞋、手套等。
  • 遵守安全操作規範，避免發生意外事故。
  • 定期進行安全教育訓練。

3. 環境保護:

  • 遵守環保法規，妥善處理廢棄物及污染物。
  • 節約能源，減少環境負擔。

4. 動線規劃:

  • 合理規劃工作動線，避免人員及物料流動產生混亂。

三、記錄與報告:

1. 設備維護記錄: 應詳細記錄所有設備的維護保養情況，包括檢查日期、維護內容、維護人員及備註等。
2. 工作環境檢查記錄: 定期進行工作環境檢查，並記錄檢查結果及改善措施。
3. 異常事件報告: 設備故障或環境異常應立即報告主管，並進行記錄。

四、持續改進:

  • 定期檢討本程序的執行情況，並根據實際情況進行調整與改進，以提升設備維護效率及工作環境品質。

附則: 本程序如有未盡事宜，依公司相關規定辦理。
