目的: 本說明書旨在提供無相應計畫事件發生時的應變處理原則，以確保人員安全、損害降至最低，並有效率地恢復正常運作。

適用範圍: 本說明書適用於所有未預先制定應變計畫的緊急事件或突發狀況。

定義:

  • 無相應計畫事件: 指任何未預先制定應變計畫，且可能造成人員傷亡、財產損失或營運中斷的事件。例如：罕見的自然災害、意外事故、重大科技故障等。

應變處理原則:

一、立即反應 (立即行動，確保安全):

1. 評估情勢: 迅速評估事件的嚴重性、影響範圍及潛在風險。
2. 確保安全: 優先保護人員生命安全，迅速疏散受威脅人員至安全區域。 採取必要的安全措施，防止事件擴大。
3. 通報相關人員: 立即通報相關主管、緊急應變小組成員（如已設立）或其他必要人員。

二、情報蒐集與確認 (了解情況，精準應變):

1. 蒐集資訊: 透過各種管道蒐集事件相關資訊，例如目擊證詞、影像資料、監控錄影等。
2. 資訊驗證: 確認資訊的可靠性，避免錯誤判斷造成不必要的損失。
3. 情勢分析: 根據蒐集到的資訊，分析事件的成因、發展趨勢及可能造成的後果。

三、應變措施執行 (積極處理，損害控制):

1. 採取初步措施: 根據情勢分析，採取必要的初步應變措施，例如控制現場、搶救傷員、防止事件擴散等。
2. 資源調配: 有效調配可用的資源，包括人力、物力、財力等，以支持應變行動。
3. 協調合作: 與相關單位（例如消防隊、警察局、醫療機構等）協調合作，共同應對事件。

四、事後處理 (檢討改善，預防再發):

1. 損害評估: 評估事件造成的損害，包括人員傷亡、財產損失、營運中斷等。
2. 事件記錄: 詳細記錄事件的過程、應變措施及結果，作為日後改進的依據。
3. 經驗總結: 從事件中吸取經驗教訓，制定或完善相關的應變計畫，以預防類似事件再次發生。
4. 檢討與改進: 檢討應變措施的有效性，並提出改進建議，以提升應變能力。
