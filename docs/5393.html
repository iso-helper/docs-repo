目的: 本程序旨在建立一套完善的有害物質管理系統，以確保公司遵守相關法規，保護員工健康與安全，並維護環境永續發展。

範圍: 本程序適用於公司所有部門及員工，涵蓋所有含有或可能含有有害物質之產品、製程、設備及廢棄物。

定義:

  • 有害物質: 指任何可能對人體健康或環境造成危害的物質，包含但不限於重金屬、揮發性有機化合物 (VOCs)、致癌物質、致畸物質、致突變物質等，以及相關法規所規範之物質。
  • SDS (Safety Data Sheet): 安全資料表，提供有害物質之理化性質、危害性、安全措施等相關資訊。
  • MSDS (Material Safety Data Sheet): (與SDS同義，舊稱) 物質安全資料表
  • 相關法規: 指政府相關單位公告之有害物質管理法規，例如：廢棄物清理法、毒性化學物質管理法、空氣污染防制法等。

程序:

一、採購階段:

1. 採購部門應於採購前，確認供應商是否提供產品之SDS/MSDS，並評估其有害物質含量是否符合公司及法規標準。
2. 應優先選用環保、低毒性或無毒性之替代產品。
3. 採購紀錄應包含產品名稱、供應商、SDS/MSDS資料、有害物質含量等資訊。

二、使用階段:

1. 各部門應妥善保管SDS/MSDS，並確保員工易於取得。
2. 使用有害物質時，應遵循SDS/MSDS所載明的安全操作程序，並配戴適當之個人防護設備 (PPE)。
3. 應建立有害物質使用記錄，包含日期、物質名稱、使用量、人員等資訊。
4. 應定期檢查設備及設施，確保其運作正常，防止有害物質洩漏或意外發生。

三、儲存階段:

1. 有害物質應儲存於通風良好、安全且標示清楚之場所。
2. 儲存場所應符合相關法規及安全規定。
3. 應定期檢查儲存場所，確保其安全及有效性。

四、廢棄階段:

1. 廢棄有害物質應依據相關法規及規定進行處理，委託合格之廢棄物處理業者處理。
2. 應妥善記錄廢棄物之種類、數量、處理方式及處理業者資訊。
3. 應避免非法排放或棄置有害物質。

五、教育訓練:

1. 公司應定期對員工進行有害物質管理之教育訓練，使其了解相關法規、安全操作程序及應變措施。
2. 教育訓練紀錄應妥善保存。

六、稽核與查檢:

1. 公司應定期進行有害物質管理之稽核與查檢，以確保本程序之有效性。
2. 稽核結果應記錄在案，並針對缺失採取改善措施。

七、應變計畫:

1. 建立應變計畫，以應對有害物質洩漏或意外事故。
2. 定期演練應變計畫，以提升員工應變能力。

記錄管理:

所有相關記錄應妥善保存，保存期限至少符合相關法規規定。

修訂:

本程序將定期檢討與修訂，以符合最新法規及公司實際情況。
