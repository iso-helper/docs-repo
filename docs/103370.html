1. 目的

本程序書旨在建立一套標準化的程序，用於識別、評估和控制組織活動對環境的潛在影響，確保符合相關的法規要求和組織的QHSE政策。

2. 適用範圍

本程序書適用於組織的所有活動，包括但不限於：營運、項目、產品和服務。

3. 責任

環境主管/經理: 負責監督和管理整個環境影響評估程序。
 項目經理/部門主管: 負責在其管轄範圍內執行環境影響評估，並確保相關措施得到落實。
所有員工: 負責識別和報告其工作活動對環境的潛在影響。

4. 程序

4.1 篩選階段:

識別潛在影響: 根據組織活動，識別可能對環境產生影響的因素，例如：空氣污染、水污染、土壤污染、廢棄物產生、噪音、振動、生態影響等。 可以使用清單、檢查表或其他工具來協助識別。
初步評估: 評估每個潛在影響的嚴重性、可能性和重要性。可以使用矩陣法或其他量化方法進行評估。 考慮相關法規要求。
 篩選: 基於初步評估結果，篩選出需要進行詳細評估的重大環境影響。

4.2 詳細評估階段:

數據收集: 收集必要的數據和信息，例如：環境基線數據、排放數據、影響範圍等。
 影響評估: 使用適當的方法對篩選出的重大環境影響進行詳細評估，例如：生命週期評估 (LCA)、風險評估等。 考慮累積效應和交互效應。
替代方案評估: 評估可行的替代方案，並比較其對環境的影響。
 制定減緩措施: 針對已識別的重大環境影響，制定有效的減緩措施，以將影響降至最低。

4.3 審查和批准階段:

文件審查: 由環境主管/經理或指定人員審查環境影響評估報告，確保其完整性、準確性和有效性。
 管理審查: 將環境影響評估報告提交給管理層審查和批准。

4.4 實施和監控階段:

實施減緩措施: 按照批准的環境影響評估報告，實施已制定的減緩措施。
 監控和測量: 定期監控和測量減緩措施的有效性，並記錄相關數據。
績效評估: 定期評估環境績效，並根據需要調整環境管理系統。
 記錄保存: 保存所有與環境影響評估相關的記錄，例如：評估報告、監測數據、減緩措施記錄等。

5. 文件控制

本程序書將由環境主管/經理控制，並定期審查和更新，以確保其與組織的QHSE政策和相關法規要求保持一致。
