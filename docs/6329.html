適用範圍: 本程序書適用於[公司名稱]所有員工及相關承攬商，涵蓋所有營運據點及相關活動。

目的: 建立一套完善的實體與環境安全管理作業程序，以確保員工、訪客及環境的安全，降低風險，並符合相關法規及標準。

責任:

  • 安全管理部門/主管: 負責制定、實施、監控及持續改進本程序書。
  • 部門主管: 負責落實本程序書於其部門，並對其部門之安全負責。
  • 所有員工及承攬商: 負責遵守本程序書的規定，並積極參與安全管理工作。

程序:

一、 實體安全管理:

1. 進出管制:

  • 所有人員進入公司場所必須登記並佩戴識別證。
  • 外來訪客需由員工陪同，並遵守訪客規範。
  • 車輛進出需遵守交通規則及相關規定，並接受檢查。
  • 夜間及非上班時間，應加強門禁管制。

2. 危險品管理:

  • 危險品儲存需符合相關法規及標準，並標示清楚。
  • 危險品操作需接受專業訓練，並遵守安全操作程序。
  • 定期檢查危險品儲存及使用情況，並記錄在案。

3. 設備安全:

  • 定期檢查及保養設備，確保其安全運作。
  • 使用設備前需接受相關訓練，並遵守操作規範。
  • 發現設備故障應立即停用並通報相關人員。

4. 消防安全:

  • 定期進行消防演習，並熟悉逃生路線。
  • 保持消防設備完好，並定期檢查及保養。
  • 發現火災應立即通報消防單位並採取滅火措施。

5. 個人防護設備 (PPE):

  • 依據工作性質提供適當的個人防護設備。
  • 員工應正確穿戴及使用個人防護設備。
  • 定期檢查個人防護設備的狀態，並及時更換損壞的設備。

二、 環境安全管理:

1. 廢棄物管理:

  • 妥善分類及處理各種類型的廢棄物。
  • 遵守相關法規及標準，並將廢棄物交由合法業者處理。
  • 定期檢查廢棄物處理情況，並記錄在案。

2. 污染防治:

  • 遵守環保相關法規及標準，防止環境污染。
  • 定期監測環境污染情況，並採取必要的防治措施。
  • 發生污染事故應立即採取應變措施，並通報相關單位。

3. 能源管理:

  • 提升能源使用效率，降低能源消耗。
  • 採用節能設備及技術。
  • 推廣節能減碳的理念。

4. 水資源管理:

  • 節約用水，避免水資源浪費。
  • 定期檢查水管及設備，防止漏水。

三、 事故應變處理:

  • 建立完善的事故應變計畫，並定期演練。
  • 發生事故應立即通報相關人員，並採取必要的應變措施。
  • 事故發生後需進行調查分析，並採取預防措施，避免再次發生。
  • 詳細記錄事故發生經過、原因、損失及處理過程。

四、 文件記錄:

  • 所有安全管理相關文件應妥善保存。
  • 定期檢視及更新安全管理相關文件。

五、 教育訓練:

  • 對所有員工進行安全教育訓練，並定期更新訓練內容。
  • 提供安全教育訓練教材，並確保員工充分理解。

六、 持續改進:

  • 定期檢討本程序書，並根據實際情況進行修訂。
  • 積極收集員工意見及建議，持續改進安全管理工作。
