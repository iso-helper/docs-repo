1. 目的

本程序書旨在建立和維護一套有效的追溯系統，以符合 FSSC 22000 食品安全管理系統的要求，確保能有效追溯產品從原料到消費者的整個流程，並能迅速且準確地識別和移除任何不符合食品安全要求的產品。

2. 範圍

本程序書適用於[公司名稱]所有生產場所，涵蓋所有生產、加工、儲存和分銷的產品。

[職稱/部門]: 負責追溯調查的執行和結果報告。
 所有員工: 負責正確執行追溯系統相關的程序。

4. 程序

4.1 原料追溯:

所有進貨原料需附帶可追溯性文件，例如供應商證明、檢驗報告、進貨單等，明確標示批號、生產日期、有效日期及相關資訊。
 原料入庫後，應按批號、日期等資訊妥善儲存並標示，確保易於追溯。
建立並維護供應商資料庫，記錄供應商資訊、產品資訊以及相關的追溯性資訊。

4.2 生產過程追溯:

所有生產過程中的半成品和成品都必須明確標示批號、生產日期、時間等資訊。
應建立生產記錄，包括生產日期、時間、原料批號、生產設備、操作人員等資訊。
 需定期校驗生產設備，並記錄校驗結果。
應記錄所有生產過程中的任何偏差或異常情況。

4.3 產品追溯:

成品出庫前，需再次確認標籤上的資訊與生產記錄一致。
建立並維護產品追溯記錄，包含產品批號、生產日期、出庫日期、客戶資訊等。
 配送過程中，應確保產品的完整性和標籤的完整性。

4.4 追溯調查:

當發生產品安全事件或客戶投訴時，應立即啟動追溯調查。
 根據追溯記錄，迅速追蹤受影響的產品批次，並採取必要的糾正措施。
將追溯調查結果記錄在案，並定期檢討以改進追溯系統。

4.5 記錄管理:

所有追溯相關記錄應妥善保存，保存期限至少為[填寫保存期限]。
記錄應清晰、完整、準確，並有追溯碼或其他可識別的標記。
 建立記錄管理系統，方便查詢和追溯。
