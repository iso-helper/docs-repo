1. 目的:

此程序的目的是明確組織內部各級管理人員在品質管理體系方面的責任和權限，以確保醫療器械符合法規要求及客戶需求。

2. 範圍:

本程序適用於所有組織人員，包括但不限於最高管理者、管理代表、部門主管及其他相關人員。

3. 責任:

3.1 最高管理者:

  • 責任:
    • 確保建立、實施和保持符合ISO 13485要求的品質管理體系。
    • 確保品質目標的設定和達成。
    • 提供必要的資源，包括財力、人力和基礎設施。
    • 審查品質管理體系的有效性。
    • 確保組織承擔其醫療器械的合規責任。
    • 促進強烈的品質文化。
    • 確保品質政策得到傳達、理解和執行。
    • 批准品質手冊。
    • 審查管理評審的輸出，並採取相應的糾正措施。
    • 確保對不符合項和糾正措施的有效處理。
    • 確保及時處理顧客抱怨。
  • 權限:
    • 批准品質政策、品質目標和資源分配。
    • 授權管理代表履行其職責。
    • 審查和批准管理評審報告。
    • 啟動必要的糾正和預防措施。

3.2 管理代表:

  • 責任:
    • 確保品質管理體系的建立、實施和保持。
    • 負責品質管理體系的運作和持續改進。
    • 監控品質管理體系的績效。
    • 確保內部審核和管理評審的有效性。
    • 向最高管理者彙報品質管理體系的績效。
    • 負責處理不符合項和糾正措施。
    • 確保對法規要求和客戶要求的符合性。
  • 權限:
    • 啟動必要的調查和糾正措施。
    • 授權相關部門和人員執行必要的活動。
    • 調動必要的資源以支持品質管理體系的運作。

3.3 部門主管:

  • 責任:
    • 確保其部門內活動符合品質管理體系要求。
    • 執行分配的品質目標。
    • 確保其部門員工接受相關培訓。
    • 參與內部審核和管理評審。
    • 負責其部門內的不符合項和糾正措施。
  • 權限:
    • 授權其部門員工執行相關職責。
    • 提出資源需求。
    • 參與品質管理體系的改進。

4. 記錄:

  • 所有與管理責任和權限相關的記錄都應保存並按照文件控制程序進行管理。

5. 程序審查:

  • 本程序應至少每年審查一次，或當出現重大變更時進行審查，以確保其持續適用性和有效性。
