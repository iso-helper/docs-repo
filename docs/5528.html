前言

本手冊旨在提供食品安全的基本知識和實務操作指南，協助您在食品處理、製作和儲存過程中，確保食品安全，預防食源性疾病的發生。 本手冊內容涵蓋個人衛生、食材處理、烹調方法、儲存方式以及清潔消毒等方面。 請務必仔細閱讀並遵守本手冊中的所有指引。

一、個人衛生

  • 洗手：這是最重要的步驟！在處理食物前後、接觸生食後、使用廁所後、處理垃圾後，都必須徹底洗手。使用肥皂和溫水至少清洗20秒，並用乾淨的毛巾或烘乾機擦乾。
  • 避免觸摸臉部、頭髮和身體：處理食物時，避免用手觸摸臉部、頭髮和身體，以免將細菌傳播到食物上。
  • 穿著乾淨衣物：穿著乾淨、合身的衣物，避免穿戴飾品 (例如戒指、手錶、項鍊等)，以免掉落食物中或成為細菌滋生的溫床。
  • 健康狀況：如有腸胃不適、發燒或其他疾病，應避免處理食物，以免將病菌傳染他人。

二、食材處理

  • 選購食材：選擇新鮮、無損壞的食材，注意包裝上的有效期限。避免購買已過期或包裝破損的食材。
  • 清潔食材：所有食材都應在烹調前徹底清洗乾淨，去除泥土、雜質和表面汙垢。 葉菜類應仔細沖洗，可浸泡於清水中數分鐘再清洗。
  • 生熟食分開：生食和熟食應分開處理，使用不同的刀具、砧板和容器，避免交叉污染。
  • 解凍食物：應以安全的方式解凍食物，例如冷藏解凍或微波爐解凍。切勿在室溫下解凍食物。
  • 避免重複使用：烹調後剩餘的食物，應妥善保存，避免重複使用。

三、烹調方法

  • 烹調溫度：食物應烹調至中心溫度達到安全標準，才能有效殺死有害細菌。例如，肉類應烹調至內部溫度達到70℃以上。
  • 徹底加熱：重新加熱食物時，應徹底加熱至冒泡，確保食物中心溫度達到70℃以上。
  • 避免交叉污染：烹調過程中，應避免生熟食交叉污染，使用乾淨的餐具和容器。

四、儲存方式

  • 冷藏溫度：易腐敗的食物應冷藏於7℃以下，以延緩細菌滋生。
  • 冷凍溫度：需要長期保存的食物應冷凍於-18℃以下。
  • 正確標示：冷凍食物應清楚標示名稱和日期，方便管理。
  • 先進先出：儲存食物應遵循「先進先出」的原則，先放入的食物應先食用。

五、清潔消毒

  • 清潔餐具：使用後應立即清洗餐具，並使用熱水和清潔劑徹底清潔，必要時可使用消毒液。
  • 清潔廚房環境：廚房環境應保持清潔衛生，定期清潔消毒檯面、爐具、水槽等。
  • 垃圾處理：垃圾應妥善處理，避免滋生害蟲和細菌。

六、常見食品安全問題及預防

  • 沙門氏菌：常見於禽類、蛋類和肉類，應徹底烹調。
  • 李斯特菌：耐低溫，易污染冷藏食品，應避免食用未經徹底烹調的冷藏食物。
  • 大腸桿菌：常見於生肉和未清洗的蔬菜，應徹底清洗食材並烹調至熟透。

七、緊急應變措施

如發生食物中毒事件，應立即就醫，並保留相關食物樣本以供檢驗。
