ISO 22000 食品安全管理系統 (FSMS) 的管理審查程序是一個系統性的過程，用於評估 FSMS 的有效性和適宜性。其目的是確保系統持續符合要求並適合其目的。 管理審查並非單純的稽核，而是更全面的評估，考慮到系統的整體表現和持續改進的機會。

ISO 22000 標準沒有規定管理審查的具體頻率或方法，但強調其重要性，並建議考慮以下因素來確定審查頻率：

  • 系統的規模和複雜性: 較大、更複雜的系統可能需要更頻繁的審查。
  • 系統的變更: 如果系統發生重大變更，則可能需要更頻繁的審查。
  • 績效指標: 審查頻率應與系統的績效指標相符。如果指標顯示需要改進，則可能需要更頻繁的審查。
  • 外部因素: 例如法規變化或供應鏈中的變化，都可能影響審查的頻率。

管理審查程序通常包括以下步驟：

1. 規劃: 確定審查的範圍、目標、參與者、時間表和資源。 這包括定義審查的範圍（哪些部分將被審查），以及需要收集的資料。

2. 資料收集: 收集有關 FSMS 績效的證據，包括：

  • 內部稽核結果: 找出不符合項和糾正措施的有效性。
  • 管理評估: 對系統整體績效的評估。
  • 監控活動的結果: 例如產品召回、客戶投訴、產品測試數據、員工培訓記錄等等。
  • 法規變動: 評估最新的法規要求是否已經被納入系統。
  • 相關方意見: 包括客戶、供應商和員工的意見反饋。

3. 審查會議: 由管理層領導的會議，審查收集的資料，評估 FSMS 的有效性和適宜性，並識別需要改進的領域。會議應包括記錄，並清晰地指出所有需要解決的問題。

4. 行動計畫: 根據審查結果制定行動計畫，以解決識別的問題並改善 FSMS。 此計劃應包括具體的行動、責任人、時間表和資源。

5. 管理評估結果的報告: 將審查結果和行動計畫正式記錄並傳達給相關部門及人員。

6. 管理評估結果的跟蹤: 追蹤行動計畫的執行情況，並確保已採取糾正和預防措施。

ISO 22000 強調管理審查應由具有決策權的管理層主持，以確保有效的資源分配和執行必要的改進。 審查結果不應僅僅是記錄，更應成為改進 FSMS 的基礎，以持續提升食品安全水平。 有效的管理審查是 ISO 22000 系統持續運作和有效性的關鍵。
