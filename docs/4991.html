1. 目的

  • 本標準作業程序書 (SOP) 的目的是規範倉儲作業流程，確保貨品安全、有效率地進出庫，並維持倉儲環境的整潔與秩序，以降低損耗及提高作業效率。

2. 適用範圍

  • 本 SOP 適用於所有進出本公司倉儲的貨品，以及所有參與倉儲作業的人員。

3. 責任

  • 倉儲管理員: 負責監督所有倉儲作業流程的執行，確保符合本 SOP 的規定。
  • 倉儲作業人員: 負責執行倉儲作業流程，並遵守本 SOP 的規定。
  • 相關部門: 負責提供正確的貨品資訊及相關文件。

4. 作業流程

4.1 進貨作業

1. 驗收: 接收到貨品後，應立即檢查貨品數量、品項、規格及品質是否與訂單相符。如有任何差異，應立即記錄並通知相關部門。
2. 登記: 將驗收結果記錄於進貨單上，並簽名確認。
3. 上架: 根據預先規劃的倉儲佈局，將貨品放置於指定位置，並確保貨品擺放整齊，方便存取。 需標示清楚貨品名稱、數量、批號及進貨日期等資訊。
4. 系統更新: 將進貨資訊更新至倉儲管理系統。

4.2 出貨作業

1. 訂單確認: 接收到出貨單後，確認訂單資訊及庫存量。
2. 揀貨: 根據出貨單上的資訊，揀選貨品。 揀貨過程中需仔細核對貨品數量及品項，避免錯誤。
3. 包裝: 將揀選的貨品進行適當的包裝，確保貨品在運輸過程中不會受損。
4. 檢查: 再次檢查貨品數量、品項及包裝是否正確。
5. 出庫登記: 將出庫資訊記錄於出貨單上，並簽名確認。
6. 系統更新: 將出貨資訊更新至倉儲管理系統。
7. 運輸安排: 安排貨品運輸，並確保貨品安全送達目的地。

4.3 庫存管理

1. 定期盤點: 定期進行庫存盤點，核對實際庫存量與系統庫存量是否相符。 盤點結果需記錄在案。
2. 庫存報表: 定期產生庫存報表，提供相關部門參考。
3. 庫存周轉: 積極管理庫存周轉率，避免庫存積壓。
4. FIFO原則: 遵循先進先出 (FIFO) 原則，避免貨品過期。

4.4 倉儲環境管理

1. 清潔: 保持倉儲環境清潔衛生，定期清理垃圾及雜物。
2. 安全: 確保倉儲環境安全，避免發生安全事故。 例如：設置消防設備、保持通道暢通等。
3. 防蟲害: 採取措施預防蟲害，保護貨品不受損害。

5. 文件管理

  • 所有倉儲相關文件，例如進貨單、出貨單、庫存報表等，應妥善保管，並按照公司規定進行歸檔。

6. 異常處理

  • 如有任何異常情況發生，例如貨品損壞、遺失或數量不符等，應立即記錄並通知相關部門處理，並追蹤處理結果。
