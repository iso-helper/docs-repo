1. 人力資源管理

1.1 人員能力與培訓:

  • 識別需求: 定期評估每個職位所需的技能、知識和經驗，並記錄在職位說明中。 這些需求將根據產品的複雜性、製造過程以及法規要求而有所不同。
  • 招聘與選擇: 採用有效的招聘和甄選程序，以確保應聘者擁有必要的資格和勝任能力。 此程序應包括面試、技能測試以及參考核實。
  • 培訓: 提供必要的培訓，以確保所有人員了解其職責、相關的法規要求以及ISO 13485標準。 培訓內容包括但不限於：
    • GMP/ISO 13485標準
    • 醫療器械相關的法規要求
    • 特定職位技能培訓
    • 文件控制程序
    • 不合格品控制程序
    • 糾正預防措施程序
  • 培訓記錄: 維護詳細的培訓記錄，包括培訓日期、內容、培訓人員以及參加人員的簽名。
  • 能力評估: 定期評估員工的能力，以確保其技能和知識符合職位要求。 這可以通過績效評估、觀察和技能測試來完成。
  • 職責分配與授權: 明確界定每個職位的職責和權限，並將其記錄在職位說明中。確保關鍵職位人員的充分授權。

1.2 人員意識:

  • 定期進行意識提升活動，以提高員工對ISO 13485標準、醫療器械相關法規以及公司品質政策的意識。 這可以通過內部培訓、會議和通告等方式來完成。

2. 基礎設施管理

2.1 設施:

  • 適宜性: 確保設施適宜於醫療器械的設計、開發、生產、儲存和分銷。 這包括考慮到潔淨度要求、環境控制、空間規劃以及設備的佈局。
  • 維護: 制定和執行設施維護計劃，以確保設施始終處於良好的工作狀態。 維護記錄應妥善保存。
  • 清潔度: 建立並實施清潔度控制程序，以防止污染和交叉污染。 此程序應包括清潔頻率、清潔方法以及清潔記錄的維護。

2.2 設備:

  • 驗證與校準: 所有設備應進行驗證和校準，以確保其精度和可靠性。 校準記錄應妥善保存。 需制定校準計畫並定期執行。
  • 維護: 制定和執行設備維護計劃，以確保設備始終處於良好的工作狀態。 維護記錄應妥善保存。
  • 設備資格: 對關鍵設備進行設備資格認證，以確保其符合預期的性能和要求。

3. 環境監控

3.1 環境條件:

  • 監控: 根據產品及生產過程的要求，監控環境條件，例如溫度、濕度、壓力、潔淨度等。 監控記錄應妥善保存。 監控頻率及方法應根據風險評估結果制定。
  • 偏差處理: 建立處理環境條件偏差的程序，並對偏差進行調查、分析和糾正。
  • 潔淨室監控: 對於需要潔淨室的生產區域，應按照相關標準進行潔淨度監控，包括粒子計數、微生物監測等。

4. 記錄管理

  • 所有與人力資源、基礎設施和環境監控相關的記錄都應按照文件控制程序妥善管理和保存。

5. 內審與管理評審

  • 本程序將定期接受內部審核和管理評審，以確保其有效性和適宜性。
