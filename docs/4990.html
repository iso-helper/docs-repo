1. 目的

  • 本標準作業程序書 (SOP) 旨在規範運輸管制流程，確保所有運輸作業安全、有效、準時且符合法規要求，降低風險，提升效率。

2. 適用範圍

  • 本 SOP 適用於所有涉及公司運輸作業的部門及人員，包括但不限於：運輸部門、倉儲部門、生產部門、採購部門等。

3. 責任與權責

  • 運輸部門主管: 負責監督和管理整個運輸流程，確保 SOP 的執行。
  • 運輸人員: 負責執行運輸作業，並遵守相關規定。
  • 倉儲部門: 負責貨物裝卸、包裝及相關作業。
  • 其他相關部門: 負責提供必要的資訊及支援。

4. 作業流程

4.1 出貨前準備:

  • 確認訂單資訊完整無誤，包含收貨人資訊、貨物數量、品名、重量、尺寸等。
  • 檢查貨物包裝是否符合標準，確保貨物安全。
  • 根據貨物特性選擇合適的運輸工具及路線。
  • 安排運輸時間及路線，並通知相關人員。
  • 進行貨物裝載前檢查，確認貨物無損壞及數量正確。
  • 完成出貨單據及相關文件。

4.2 運輸過程:

  • 運輸人員應遵守交通規則及相關法規。
  • 運輸過程中應定期檢查貨物狀況，確保貨物安全。
  • 遇有緊急情況，應立即採取應變措施並通報相關部門。
  • 準時抵達目的地，並完成貨物卸載。

4.3 到貨確認:

  • 收貨人確認貨物數量、品名、規格及狀態。
  • 完成簽收手續，並提交相關文件。
  • 處理貨物異常情況，如損壞、遺失等。

4.4 文件管理:

  • 所有運輸相關文件應妥善保存，並依規定保存期限。
  • 文件應清晰、完整，並符合公司規定。

5. 緊急應變措施

  • 交通事故: 立即報警，並通知公司相關部門，配合警方調查。
  • 貨物損壞: 拍照記錄損壞情況，並立即通報相關部門，進行損失評估及處理。
  • 貨物遺失: 立即報警，並通知公司相關部門，追查遺失貨物。
