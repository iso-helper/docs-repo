目的: 建立一套標準程序，用以有效矯正安全衛生異常事件，並預防其再次發生，以確保工作場所安全與健康。

適用範圍: 本程序適用於所有可能導致安全衛生事故的異常事件，包含但不限於：

  • 近失事故 (Near Miss)
  • 輕微事故
  • 重大安全衛生事故
  • 潛在危險因素

程序:

一、異常事件的通報與調查:

1. 立即通報: 任何人員發現安全衛生異常事件，應立即向其主管或指定人員通報。通報應包含事件時間、地點、人員、事件經過及初步觀察結果等。
2. 現場保留: 盡可能保留事件現場原狀，除非保留現場會造成更危險的情況。
3. 初步調查: 由主管或指定人員進行初步調查，收集相關證據，例如照片、影片、證詞等。
4. 事故調查報告: 根據調查結果，撰寫事故調查報告，內容應包含：

  • 事件描述
  • 事故原因分析 (包含人為因素、設備因素、環境因素等)
  • 受傷或損害程度
  • 證人證詞
  • 相關圖片或影片

5. 報告提交: 將事故調查報告提交至相關部門審核及記錄。

二、異常事件的矯正措施:

1. 短期矯正措施: 針對立即性的危險，採取必要的短期矯正措施，以確保現場安全。
2. 根本原因分析: 針對事故根本原因進行深入分析，找出問題的根源。
3. 長期矯正措施: 根據根本原因分析結果，制定有效的長期矯正措施，預防類似事件再次發生。 此措施應包含：

  • 改善作業流程
  • 提升設備及設施
  • 加強員工訓練
  • 制定新的安全規範或修訂現有規範

4. 措施執行: 指派相關人員執行矯正措施，並設定完成期限。
5. 措施驗證: 矯正措施執行完成後，應進行驗證，確保其有效性。

三、預防措施:

1. 風險評估: 定期進行風險評估，識別潛在的危險因素。
2. 安全教育訓練: 針對相關人員進行安全教育訓練，提升安全意識。
3. 安全宣導: 透過各種管道進行安全宣導，提高員工的安全意識。
4. 安全檢查: 定期進行安全檢查，及早發現並處理潛在危險。
5. 持續改善: 持續改進安全管理系統，以減少安全衛生事故的發生。

四、記錄與追蹤:

1. 事故記錄: 所有安全衛生異常事件均應詳細記錄，並建立資料庫。
2. 矯正措施追蹤: 追蹤矯正措施的執行進度及效果。
3. 定期檢討: 定期檢討本程序的有效性，並進行必要的修訂。

五、相關責任:

  • 各級主管負責監督員工遵守安全規範，並及時處理安全衛生異常事件。
  • 員工有責任遵守安全規範，並積極參與安全衛生管理。
  • 安全衛生管理部門負責制定和完善安全衛生管理制度，並監督執行。

備註: 本程序應根據實際情況進行調整和完善。
