1. 目的

  • 本程序書旨在確立執行ISO 50001 能源管理系統管理審查的程序，以確保能源管理系統的有效性、適宜性和持續改進。

2. 範圍

  • 本程序書適用於[公司名稱]所有從事能源管理系統運作的部門及人員。

3. 參考文件

  • ISO 50001 能源管理系統
  • [公司名稱]能源管理手冊
  • [其他相關文件，例如：內部程序書]

4. 定義

  • 管理審查: 系統性地評估能源管理系統的績效，以確認其持續符合性、充分性及有效性。
  • 管理代表: 負責能源管理系統整體運作的人員。

5. 程序

5.1 審查規劃:

  • 頻率: 管理審查至少每年進行一次，或根據組織的規模、複雜性和變化的頻率調整。
  • 審查範圍: 包括能源管理系統所有要素，包括能源績效指標(EnPI)、能源使用數據、能源效率措施的執行情況、符合法規及其他要求的情況以及持續改進的機會。
  • 審查團隊: 由管理代表組成審查團隊，成員應具備相關知識和經驗。 可邀請相關部門主管、能源管理人員以及其他相關人員參與。
  • 審查時間及地點: 預先安排審查時間及地點，並提前通知相關人員。
  • 審查資料準備: 在審查前，需收集並準備相關文件，例如能源績效指標數據、能源消耗數據、能源審計報告、內部稽核報告、相關會議記錄、管理文件記錄等。

5.2 審查執行:

  • 開幕會議: 審查團隊召開開幕會議，簡述審查目的、範圍及程序。
  • 資料評估: 審查團隊評估準備好的資料，確認能源管理系統的有效性。
  • 現場觀察 (視情況而定): 審查團隊可進行現場觀察，以驗證記錄的準確性和系統的實際運作情況。
  • 訪談: 審查團隊可與相關人員進行訪談，收集額外信息。
  • 分析及評估: 審查團隊對收集到的資料進行分析和評估，識別能源管理系統的優勢、不足之處、機會和威脅(SWOT 分析)。

5.3 審查結果報告:

  • 記錄: 審查團隊應將審查結果記錄在管理審查報告中，包括識別出的不符合項、糾正措施、改進機會和行動計畫。
  • 分發: 管理審查報告應分發給相關人員，包括高階管理層。

5.4 糾正措施及持續改進:

  • 執行: 根據審查結果，制定糾正措施和預防措施，並分配責任人及期限。
  • 追蹤: 追蹤糾正措施和預防措施的執行情況，並定期檢查其有效性。
  • 持續改進: 基於管理審查的結果，持續改進能源管理系統。

5.5 文件管理:

  • 所有與管理審查相關的文件，例如審查計畫、審查報告、糾正措施記錄等，應妥善保存和管理。

6. 相關職責

  • 管理代表: 負責規劃和組織管理審查，確保審查的有效性。
  • 審查團隊成員: 負責參與審查，並提供專業意見。
  • 相關部門主管: 負責提供相關資料，並配合審查工作。
