一、 招募與甄選

1.1 招募規劃:

  • 確定職位需求與職位說明書撰寫。
  • 擬定招募策略 (例如：內部招募、外部招募、徵才管道)。
  • 制定招募預算。

1.2 招募執行:

  • 發布職位廣告 (包含平台選擇、廣告文案撰寫)。
  • 收集應徵資料。
  • 篩選應徵者履歷。

1.3 甄選流程:

  • 面試安排與執行 (包含面試題目設計、評分標準)。
  • 能力測驗或技能評估 (視職位需求)。
  • 背景調查 (包含參考查核、學歷驗證)。

1.4 錄取與錄用:

  • 錄取通知。
  • 錄用合約簽訂。
  • onboarding (新人訓練及適應期規劃)。

二、 績效管理

2.1 績效目標設定:

  • 設定 SMART 目標 (Specific、Measurable、Achievable、Relevant、Time-bound)。
  • 與員工共同討論並確認績效目標。

2.2 績效評估:

  • 定期績效考核 (例如：年度考核、季考核)。
  • 使用客觀評估標準。
  • 提供績效回饋。

2.3 績效改進:

  • 制定績效改進計畫。
  • 提供必要的培訓與支持。
  • 追蹤績效改進情況。

三、 薪資福利

3.1 薪資結構設計:

  • 薪資級別設定。
  • 薪資調整機制。
  • 獎金制度設計。

3.2 福利規劃:

  • 法定福利 (例如：勞健保、退休金)。
  • 員工福利 (例如：員工旅遊、健康檢查)。
  • 福利制度宣導。

四、 訓練與發展

4.1 訓練需求分析:

  • 評估員工技能差距。
  • 識別訓練需求。

4.2 訓練規劃與執行:

  • 設計訓練課程。
  • 選擇訓練方式 (例如：內訓、外訓、線上學習)。
  • 執行訓練課程。

4.3 訓練評估:

  • 評估訓練成效。
  • 根據評估結果調整訓練計畫。

五、 員工關係管理

5.1 溝通機制:

  • 建立有效的溝通管道 (例如：員工座談會、定期會議)。
  • 及時處理員工問題與疑慮。

5.2 勞資關係:

  • 遵守勞動相關法規。
  • 建立良好的勞資關係。
  • 處理勞資爭議。

5.3 員工離職管理:

  • 辦理離職手續。
  • 進行離職面談。
  • 進行離職後之追蹤。

六、 法令遵循

  • 遵守所有相關的勞動法規及政府規定。
  • 定期更新並檢視人力資源管理程序，以符合最新的法規。

七、 文件管理

  • 所有人力資源相關文件應妥善保存並管理，確保資料安全與保密。
