目的: 建立一套標準化的客戶服務流程，提升客戶滿意度，有效解決客戶問題，並持續改善服務品質。

 

適用範圍: 所有與客戶互動的部門及人員。

程序:

一、客戶服務接洽:

1. 多管道接洽: 提供多種管道讓客戶聯繫，例如電話、電子郵件、線上聊天、社群媒體等，並確保各管道皆有明確的聯繫方式及回應時間。
2. 即時回應: 盡可能在短時間內回應客戶的詢問或投訴，並明確告知處理進度。電話接聽應在三聲內接起，線上回覆應在工作時間內一小時內回覆。
3. 專業禮貌: 所有客服人員應以專業、禮貌的態度與客戶溝通，使用親切的語氣，並展現積極解決問題的意願。
4. 主動傾聽: 仔細聆聽客戶的陳述，並確認已理解客戶的需求和問題。避免打斷客戶說話，並適時提問以釐清細節。
5. 身份驗證: 根據公司規定，對客戶進行必要的身份驗證，以保障客戶資訊安全。

二、問題處理:

1. 問題分類: 將客戶問題分類，以便有效分配資源及追蹤處理進度。
2. 解決方案查找: 根據問題類型，參考既有的解決方案資料庫或相關文件，盡力提供快速有效的解決方案。
3. 專業知識應用: 客服人員應具備必要的專業知識，才能有效解決客戶問題。必要時應尋求相關部門協助。
4. 記錄追蹤: 所有客戶互動及問題處理過程均需詳細記錄，包括客戶資訊、問題描述、處理步驟、解決結果及處理時間等。
5. 權限管控: 根據不同問題的複雜程度，設定不同處理人員的權限，確保問題能得到妥善處理。

三、客戶投訴處理:

1. 積極回應: 迅速回應客戶投訴，並表達歉意。
2. 深入調查: 對投訴內容進行深入調查，了解事件的來龍去脈。
3. 提出解決方案: 根據調查結果，提出合理的解決方案，並與客戶協商。
4. 持續追蹤: 持續追蹤解決方案的執行情況，並確保客戶滿意。
5. 回饋機制: 將客戶投訴作為改進服務的機會，並建立回饋機制，持續提升服務品質。

四、服務品質監控:

1. 客戶滿意度調查: 定期進行客戶滿意度調查，了解客戶對服務的評價。
2. 績效考核: 將客戶服務績效納入客服人員的績效考核指標。
3. 流程優化: 根據客戶意見回饋及績效考核結果，持續優化客戶服務流程。
4. 知識庫更新: 定期更新解決方案資料庫及相關文件，以確保資訊的準確性和完整性。

五、相關責任:

  • 客服部門主管: 負責監督及管理整個客戶服務流程的執行。
  • 客服人員: 負責執行客戶服務流程，並提供高品質的客戶服務。
  • 相關部門: 負責提供必要的支援和協助，以確保客戶服務流程的順利執行。
