1. 目的

  • 本程序書旨在建立和維持文件控制系統，確保所有文件保持最新、準確、適用且易於存取，以符合ISO 50001能源管理系統的要求。

2. 範圍

本程序書適用於所有與能源管理系統相關的文件，包括但不限於：

  • 能源管理手冊
  • 能源績效指標（KPI）
  • 能源審查記錄
  • 能源效率措施（EEM）執行計劃
  • 能源消耗數據記錄
  • 法規及標準文件
  • 程序文件
  • 指南文件
  • 表格
  • 圖表

3. 職責

  • 能源管理代表 (EnMS 代表): 負責文件控制系統的建立、實施和維持，並確保其有效性。
  • 部門主管/相關人員: 負責其部門相關文件的建立、更新、審查和控制。
  • 文件管理員 (如有): 負責文件的建立、編號、分發、儲存和銷毀。

4. 程序

4.1 文件的建立與審查:

  • 所有新文件在建立前，需經過相關部門主管的審查和批准。
  • 文件需明確標示文件編號、版本號、發布日期、修訂日期、作者、批准者及頁數。
  • 文件內容需清晰、準確、完整且易於理解。
  • 文件審查頻率需根據文件的性質和重要性而定，並記錄審查結果。

4.2 文件的批准與發布:

  • 經審查批准的文件，需由授權人員簽署批准。
  • 批准的文件需發布給相關人員，並確保其獲得及使用最新版本的相關文件。
  • 發布方式可以包含但不限於電子郵件、內網公告、文件管理系統等。

4.3 文件的發放與控制:

  • 文件的發放應控制在需要的人員範圍內。
  • 應建立文件發放記錄，追蹤文件的發放情況。
  • 使用電子文件管理系統可以提高文件的可追溯性及版本控制。

4.4 文件的修訂與更新:

  • 當文件內容需要修改時，應建立新的版本號，並標明修訂日期及修訂內容。
  • 修訂文件需經過審查和批准程序，並通知相關人員。
  • 舊版本文件應明確標記為「作廢」，並妥善保存備份。

4.5 文件的儲存與備份:

  • 文件應妥善儲存，確保其完整性及安全性。
  • 應建立文件儲存目錄，並定期整理及更新。
  • 應定期備份重要的文件，防止文件遺失或損壞。

4.6 文件的銷毀:

  • 作廢的文件應根據既定的程序銷毀，並記錄銷毀過程。
  • 銷毀方式應符合公司相關規定及法規要求。

5. 記錄

以下記錄需妥善保存：

  • 文件審查記錄
  • 文件批准記錄
  • 文件發放記錄
  • 文件修訂記錄
  • 文件銷毀記錄
