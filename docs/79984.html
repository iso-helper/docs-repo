1. 樣品採集 (Sampling):

樣品計畫: 制定明確的樣品計畫，包含採樣頻率、樣品數量、採樣位置、採樣方法等。 計畫應基於風險評估，針對高風險產品或製程應增加採樣頻率和數量。
 採樣程序: 建立標準操作程序 (SOP)，詳細描述採樣步驟，確保採樣過程的科學性和一致性，避免樣品汙染或變質。 SOP 應包含：
樣品識別和標籤：明確標示樣品編號、採樣日期、時間、位置、批號等資訊。
 樣品容器和保存：使用合適的容器和保存條件（溫度、濕度等）以保持樣品完整性。
採樣工具的清潔和消毒：確保採樣工具乾淨無菌，避免交叉污染。
 採樣人員的培訓：採樣人員應接受充分的培訓，了解採樣程序和注意事項。
樣品記錄: 完整記錄所有採樣相關資訊，包括樣品計畫、採樣記錄、異常情況等。

2. 測試方法 (Testing Methods):

方法驗證/確認: 選擇合適的測試方法，並進行方法驗證或確認，確保方法的準確性、精密度、特異性等符合要求。
標準操作程序 (SOP): 建立詳細的測試方法 SOP，包含測試步驟、儀器設備、試劑、計算公式、結果判讀標準等。
 儀器校準和維護: 定期校準和維護測試儀器設備，確保其準確性和可靠性。 建立儀器校準和維護記錄。
試劑和標準品管理: 妥善管理試劑和標準品，確保其品質和有效期。

3. 結果記錄 (Result Recording):

數據完整性: 確保測試數據的完整性、準確性和可追溯性。
數據記錄系統: 使用電子數據記錄系統 (EDRS) 或其他合適的數據記錄系統，避免手寫記錄的錯誤和模糊性。
 原始數據記錄: 保留原始數據記錄，並妥善保管。
數據審核: 對測試數據進行審核，確保數據的正確性和一致性。

4. 偏差處理 (Deviation Handling):

偏差報告: 建立偏差報告程序，對任何與預期結果不符的情況進行報告。
偏差調查: 對偏差進行調查，確定偏差的原因，並評估其影響。
 糾正措施: 採取糾正措施，消除偏差，防止偏差再次發生。
偏差記錄: 完整記錄偏差報告、調查、糾正措施等相關資訊。

5. CAPA (糾正預防措施) (Corrective and Preventive Actions):

CAPA 系統: 建立完善的 CAPA 系統，對偏差、不合格品、客戶投訴等進行調查分析，並制定糾正和預防措施。
風險評估: 對 CAPA 的有效性進行評估，並對風險進行評估。
 CAPA 的實施和驗證: 確保 CAPA 的有效實施和驗證。
CAPA 記錄: 完整記錄 CAPA 的所有步驟和結果。

額外考量:

文件控制: 所有與 GMP 品質控制程序相關的文件應進行有效的控制，確保文件版本的正確性和一致性。
培訓: 所有參與 GMP 品質控制程序的人員應接受充分的培訓。
 內審和管理審查: 定期進行內審和管理審查，評估 GMP 品質控制程序的有效性和合規性。
