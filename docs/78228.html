此程序描述了根據 ISO 13485 標準進行內部稽核的規劃、執行和報告流程。

一、 規劃階段:

1. 稽核範圍和目標的確定: 確定本次內部稽核的範圍（例如，特定部門、流程、產品或系統），並設定明確且可衡量的稽核目標。 這應基於風險評估結果，優先稽核高風險領域。 應明確說明要稽核的標準要求。

2. 稽核計劃的制定: 制定詳細的稽核計劃，包括：

  • 稽核時間表: 明確稽核日期、時間和持續時間。
  • 稽核組成員: 指派具有適當資格和經驗的稽核員，並明確稽核組長。 避免稽核員審查其自身的工作。
  • 稽核清單/檢查表: 根據 ISO 13485 標準要求及稽核範圍，編制稽核清單或檢查表，以確保所有相關要求均得到覆蓋。 清單應包含具體的稽核項目和評估標準。
  • 資源分配: 確保稽核活動所需的資源，包括文件、設備和時間。
  • 溝通計劃: 確定如何與被稽核部門溝通稽核計劃和結果。

3. 稽核員培訓: 確保所有稽核員都接受過 ISO 13485 標準、稽核技巧和相關法規要求的培訓，並具備執行稽核所需的技能和知識。

4. 文件審閱: 在稽核前，稽核員應審閱相關的文件，例如質量手冊、程序文件、記錄等，以了解被稽核部門的運作情況。

二、 執行階段:

1. 稽核開展會議: 在稽核開始前，與被稽核部門的管理人員進行開展會議，說明稽核目的、範圍和程序。

2. 證據收集: 稽核員應通過觀察、訪談、檢查記錄、檢驗產品等方式收集客觀證據，以評估被稽核部門是否符合 ISO 13485 標準要求。

3. 不符合項的識別: 稽核員應識別任何不符合 ISO 13485 標準要求或公司內部程序的情況，並詳細記錄。 應記錄不符合項的證據、嚴重程度和影響。 區分觀察、不符合項和重大不符合項。

4. 證據的評估: 稽核員應對收集的證據進行評估，以確定其是否足以支持稽核結論。

5. 糾正和預防措施的討論: 與被稽核部門討論不符合項，並共同探討糾正和預防措施。

三、 報告階段:

1. 稽核報告的編制: 稽核結束後，稽核員應編制一份詳細的稽核報告，包括：

  • 稽核範圍和目標: 簡要描述稽核的範圍和目標。
  • 稽核日期和持續時間: 明確稽核的時間。
  • 稽核組成員: 列出所有參與稽核的成員。
  • 稽核發現: 詳細描述所有觀察、不符合項和重大不符合項，包括證據、嚴重程度和影響。
  • 糾正和預防措施: 描述被稽核部門計劃採取的糾正和預防措施。
  • 稽核結論: 總結稽核結果，評估被稽核部門的符合性水平。
  • 稽核員簽名: 稽核報告應由稽核員簽名。

2. 報告的發布和審閱: 稽核報告應提交給管理層審閱和批准。

3. 管理評審: 稽核報告的發現和建議應在管理評審中進行討論，並制定相應的糾正和預防措施。

4. 糾正措施的驗證: 應對已實施的糾正措施進行驗證，以確保其有效性。

持續改進: 此內部稽核程序應定期評估和更新，以確保其有效性和適宜性。 稽核結果應用於持續改進質量管理體系。
