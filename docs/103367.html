1. 目的

本程序書旨在建立一套完善的危險性工作許可證 (Permit-to-Work, PTW) 系統，以確保所有高風險作業均在安全可控的條件下進行，有效預防和控制事故的發生，保障人員安全和環境保護，符合QHSE（品質、健康、安全、環境）管理體系的要求。

2. 適用範圍

本程序書適用於所有涉及高風險作業的活動，包括但不限於：

高處作業
 受限空間作業
熱作業 (例如焊接、切割、熱處理)
 電氣作業
管道作業
 化學品操作
機械設備維修保養
 爆炸性環境作業
其他經風險評估判定為高風險的作業

3. 責任與權限

作業主管/領班: 負責完成工作許可證申請、執行工作許可證上的安全措施、監督作業人員執行安全措施、並在作業完成後關閉許可證。
安全管理人員/部門: 負責審批工作許可證、監督工作許可證制度的執行、提供安全技術指導、進行風險評估和安全培訓。
 作業人員: 負責遵守工作許可證上的規定，並在發現任何安全隱患時立即報告。

4. 程序步驟

(一) 工作許可證申請:

1. 作業主管/領班根據作業內容，進行風險評估，並根據風險評估結果，確定是否需要申請工作許可證。
2. 填寫工作許可證申請表，詳細說明作業內容、地點、時間、涉及的危險因素、所需的控制措施、所需的資源和人員等。
3. 將申請表提交給安全管理人員/部門審批。

(二) 工作許可證審批:

1. 安全管理人員/部門審核工作許可證申請表，確認風險評估的充分性以及控制措施的有效性。
2. 如有需要，安全管理人員/部門可要求申請人提供更詳細的信息或修改申請內容。
3. 審批通過後，安全管理人員/部門在工作許可證上簽字批准。

(三) 工作許可證執行:

1. 作業主管/領班向所有參與作業的人員宣讀工作許可證上的安全規定。
2. 作業人員必須遵守工作許可證上的安全規定，並在作業過程中採取必要的安全措施。
3. 作業主管/領班必須監督作業人員的執行情況，並及時糾正違規行為。
4. 作業過程中，如遇任何異常情況，應立即停止作業，並向安全管理人員/部門報告。

(四) 工作許可證關閉:

1. 作業完成後，作業主管/領班應檢查作業現場，確保所有安全隱患已消除。
2. 在工作許可證上簽字，並註明關閉時間。
3. 將關閉的工作許可證歸檔保存。

5. 工作許可證的類型與格式:

工作許可證應根據不同作業類型設計不同的格式，至少應包括以下內容：

  • 工作許可證編號
  • 作業名稱
  • 作業地點
  • 作業時間
  • 作業人員
  • 危險因素識別
  • 控制措施
  • 應急預案
  • 申請人簽名
  • 審批人簽名
  • 作業開始時間
  • 作業結束時間
  • 關閉人簽名

6. 文件記錄與管理:

所有工作許可證申請表、審批表和關閉記錄應妥善保存，並建立完整的檔案管理制度，方便查閱和追溯。

7. 培訓與教育:

所有涉及高風險作業的人員都必須接受相關的安全培訓，了解工作許可證制度的內容和操作流程。

8. 定期審查與更新:

本程序書應定期審查和更新，以確保其符合最新的安全法規和公司要求。
