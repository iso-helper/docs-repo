1. 目的

本程序文件描述了本公司根據 ISO 22716:2007 (良好製造規範：化妝品) 的要求，建立、實施和維護糾正措施和預防措施 (CAPA) 系統的程序，用以識別、調查、糾正偏差，並預防其再次發生。 此系統旨在持續改進我們的品質管理體系，確保產品符合法規要求和顧客期望。

2. 範圍

本程序適用於所有可能影響產品品質、安全和符合法規要求的偏差，包括但不限於：

製程偏差
 產品不合格
客戶投訴
 審計發現
法規不符合

3. 責任

品質管理部門: 負責建立和維護 CAPA 系統，並對 CAPA 程序的有效性進行監控。
部門主管: 負責在各自的部門內識別、調查並糾正偏差。
 相關人員: 負責根據其職責參與 CAPA 的調查、糾正和預防措施的實施。

4. 程序

4.1 偏差的識別和報告:

所有員工有責任識別並及時報告任何可能影響產品品質、安全和符合法規要求的偏差。
 偏差應使用預先指定的報告表格記錄，並包含偏差的日期、時間、地點、描述、影響和相關人員等信息。
報告的偏差應由部門主管或指定人員進行評估，以確定其嚴重性。

4.2 偏差的調查:

針對所有已報告的偏差，應進行徹底的調查，以確定根本原因。
調查應包括：
 收集數據和證據；
訪談相關人員；
 評估風險；
分析根本原因。
 調查結果應以書面形式記錄，並包含根本原因分析、證據和相關結論。

4.3 糾正措施:

基於調查結果，應制定並實施糾正措施，以消除偏差的直接原因。
 糾正措施應明確、可衡量、可實現、相關和有時限 (SMART)。
糾正措施的實施情況應進行監控，並記錄驗證結果。

4.4 預防措施:

為防止偏差再次發生，應制定並實施預防措施。
預防措施應針對根本原因，以防止類似偏差的發生。
 預防措施的有效性應進行評估和監控。

4.5 CAPA 的審查:

定期（例如，每年）對 CAPA 系統的有效性進行審查，以確保其符合 ISO 22716 的要求。
 審查應評估 CAPA 系統的績效，並識別改進的機會。
審查結果應記錄在案。

4.6 記錄管理:

所有與 CAPA 相關的記錄應妥善保存，並根據公司記錄管理程序進行管理。
記錄應易於訪問和追溯。
