目的: 建立一套標準化的檢驗管理程序，確保檢驗工作的準確性、一致性和效率，以滿足產品品質要求及法規規範。

適用範圍: 本程序適用於所有需要進行檢驗的產品、材料、設備及過程。

責任:

  • 品管部門主管: 負責制定、修訂及執行本程序，並監督檢驗工作的進行。
  • 檢驗人員: 負責按照本程序執行檢驗工作，並記錄檢驗結果。
  • 生產部門: 負責提供檢驗所需的樣品及相關資料。

程序:

一、 檢驗計劃:

1. 制定檢驗計劃: 根據產品規格、客戶要求及法規規範，制定詳細的檢驗計劃，包括檢驗項目、檢驗方法、檢驗頻率、樣品數量及合格標準等。 檢驗計劃應由品管部門主管審核批准。
2. 樣品採樣: 按照規定的樣品採樣方法，從生產線上或倉庫中採集具有代表性的樣品。 樣品採集過程應記錄在案，並確保樣品完整性及可追溯性。
3. 樣品標識: 所有樣品均需清楚標識，包括樣品編號、採樣日期、生產批號、產品名稱等。

二、 檢驗執行:

1. 檢驗環境: 檢驗工作應在符合相關規範的環境下進行，確保檢驗結果的準確性。
2. 檢驗方法: 檢驗人員應按照規定的檢驗方法執行檢驗工作，並使用經過校準的儀器設備。
3. 檢驗記錄: 檢驗人員應如實記錄檢驗過程及結果，包括檢驗日期、檢驗人員、檢驗項目、檢驗方法、檢驗結果及異常情況等。 記錄應清晰、完整、可追溯。
4. 偏差處理: 若檢驗結果超出合格標準，應立即停止生產，並根據既定的偏差處理程序進行處理，例如：隔離產品、調查原因、採取糾正措施等。

三、 檢驗結果審核:

1. 結果評估: 品管部門主管應審核檢驗記錄及結果，確認檢驗工作的完整性和準確性。
2. 報告編制: 根據檢驗結果編制檢驗報告，並將報告提交相關部門。
3. 資料保存: 檢驗記錄及報告應妥善保存，以便追溯及參考。

四、 設備管理:

1. 儀器校準: 所有用於檢驗的儀器設備應定期進行校準，並保持校準記錄。
2. 設備維護: 應定期維護檢驗設備，確保其正常運作。

五、 程序修訂:

  • 本程序應定期進行審查和修訂，以確保其符合最新的規範和要求。
