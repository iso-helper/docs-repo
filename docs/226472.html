ISO 21001:2018 的非符合項管理程序旨在識別、記錄、分析和糾正任何與標準要求不符的情況。 一個有效的程序應包含以下步驟：

1. 非符合項的識別和記錄:

主動識別: 透過內部審核、管理評審、顧客回饋、監控與測量等活動，主動辨識非符合項。
被動識別: 透過客戶投訴、事故報告、檢驗結果等方式發現非符合項。
記錄: 所有非符合項必須記錄在一個明確的系統中。記錄應包含以下資訊：
非符合項的唯一識別編號
非符合項的性質
發現非符合項的日期和時間
發現非符合項的責任者
涉及的教育培訓活動或學習材料
影響範圍
相關的證據（例如照片、測試報告）

2. 非符合項的分析:

根本原因分析: 確定非符合項的根本原因，而不是只專注於症狀。可以使用各種工具，例如魚骨圖、5 Why 分析等。
風險評估: 評估非符合項對教育服務的影響和潛在風險。

3. 糾正措施的發展與實施:

糾正措施: 制定有效的糾正措施以消除非符合項，防止其再次發生。措施應具體、可測量、可實現、相關和有時限（SMART原則）。
實施: 實施制定的糾正措施，並記錄實施過程。
驗證: 驗證糾正措施的有效性，確保問題得到解決。

4. 預防措施的製定與實施:

預防措施: 制定預防措施，以防止類似非符合項的再次發生。 這需要關注系統性問題，而非只是解決個案。
實施: 實施制定的預防措施，並記錄實施過程。
驗證: 驗證預防措施的有效性。

5. 管理評審:

對非符合項的管理過程進行定期審查，評估其有效性，並識別改進的機會。

6. 記錄保存:

所有與非符合項管理相關的記錄都應妥善保存，並符合組織的記錄管理程序。

7. 持續改進:

* 透過非符合項目的分析和糾正措施的實施，不斷改善教育服務管理體系。

範例流程圖:

[開始] -> [識別非符合項目] -> [記錄非符合項目] -> [分析根本原因] -> [制定糾正措施] -> [實施糾正措施] -> [驗證糾正措施] -> [制定預防措施] -> [實施預防措施] -> [驗證預防措施] -> [管理審查] -> [持續改進] -> [結束]
