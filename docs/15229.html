目的: 建立一套標準作業程序，用以系統性地鑑別工作場所潛在危害，評估其風險，並制定有效的控制措施，以降低職業災害發生的機率及嚴重程度。

適用範圍: 本程序適用於本公司所有工作場所及作業活動，包含但不限於生產作業、維護保養、辦公室作業等。

程序:

一、危害鑑別:

1. 組成危害鑑別團隊: 團隊成員應包含具備相關專業知識及經驗之人員，例如：安全衛生管理人員、主管、作業人員等。
2. 選擇鑑別方法: 可採用以下方法，或其組合：

  • 查檢表法: 使用預先設計好的查檢表，系統性地檢查潛在危害。
  • 腦力激盪法: 團隊成員共同討論，提出潛在危害。
  • 流程圖法: 以流程圖呈現作業流程，找出每個步驟中的潛在危害。
  • 觀察法: 直接觀察工作場所及作業活動，辨識潛在危害。
  • 事故資料分析法: 分析過去事故資料，找出潛在危害。
  • 專家諮詢法: 尋求專業人士的意見，協助鑑別潛在危害。

3. 記錄危害資訊: 將鑑別出的危害詳實記錄，包含危害來源、型態、可能發生的時間、地點及影響範圍等。 使用表格或其他適當方式記錄。
4. 危害分類: 依據危害的型態或性質，將其分類，例如：物理性危害 (噪音、振動、輻射)、化學性危害 (毒性物質、腐蝕性物質)、生物性危害 (細菌、病毒)、人因性危害 (疲勞、壓力)、心理社會危害 (霸凌、性騷擾) 等。

二、風險評估:

1. 風險評估方法: 使用風險矩陣或其他風險評估方法，例如：危險度評估、層級評估法等。 選擇的方法需符合公司實際情況和法規要求。
2. 風險評估要素: 評估時需考慮以下要素：

  • 危害的嚴重性 (Severity): 危害發生可能造成的傷害程度，例如：輕微傷害、重傷、死亡。
  • 危害發生的可能性 (Likelihood/Probability): 危害發生的機率，例如：低、中、高。

3. 風險等級判定: 根據危害的嚴重性和可能性，判定風險等級，例如：低、中、高、極高。 可使用風險矩陣圖表化呈現。
4. 記錄風險評估結果: 將評估結果詳實記錄，包含危害名稱、嚴重性、可能性、風險等級、及評估日期等。

三、風險控制:

1. 制定控制措施: 根據風險等級，制定適當的控制措施，依據危害控制層級原則 (消除、替代、工程控制、行政控制、個人防護具)，選擇最有效的控制方法。
2. 控制措施評估: 評估控制措施的有效性，確保其能有效降低風險。
3. 記錄控制措施: 將所有控制措施詳實記錄，包含措施內容、執行者、執行日期及預期效果等。

四、監控與檢討:

1. 定期監控: 定期監控控制措施的有效性及執行情況，並記錄監控結果。
2. 檢討與更新: 定期檢討危害鑑別與風險評估結果，並根據實際情況更新作業程序及控制措施。 至少每年檢討一次，或有重大變更時立即檢討。

五、文件管理:

  • 所有危害鑑別、風險評估及控制措施相關文件應妥善保存，並依據公司文件管理規定進行管理。
