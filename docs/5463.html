目的: 建立一套完善的內部稽核管理程序，以確保公司營運符合相關法規、政策、程序及內部控制制度，並提升營運效率及風險管理效能。

適用範圍: 本程序適用於公司所有部門及單位，涵蓋所有內部稽核活動。

責任:

  • 稽核委員會 (如有): 監督內部稽核部門之運作，審閱稽核報告，並向董事會報告。
  • 內部稽核主管: 負責規劃、執行及監督內部稽核計畫之執行，確保稽核工作之獨立性、客觀性及專業性。
  • 內部稽核人員: 負責執行稽核計畫，蒐集稽核證據，撰寫稽核報告，並追蹤管理階層之改善行動。
  • 各部門主管: 負責配合內部稽核工作，提供相關資料及協助，並落實稽核發現事項之改善行動。

程序:

一、 稽核計畫的規劃與制定:

1. 風險評估: 每年至少進行一次全面的風險評估，識別並評估公司營運中可能存在的重大風險，作為擬定稽核計畫的依據。 風險評估應考慮營運環境變遷、新科技應用、法規變動等因素。
2. 稽核計畫的制定: 根據風險評估結果，制定年度稽核計畫，明確稽核範圍、目標、時間表及所需資源。 計畫應包含不同領域的稽核，以確保稽核覆蓋範圍充分。
3. 資源配置: 根據稽核計畫，分配必要的資源，包括人力、時間、預算及技術設備。
4. 稽核計畫審查: 稽核計畫應經由內部稽核主管及相關主管審查核准。

二、 稽核工作的執行:

1. 稽核取樣: 採用適當的取樣方法，確保稽核結果具有代表性及可靠性。
2. 證據蒐集: 使用多種證據蒐集方法，例如：訪談、文件審查、觀察及分析。
3. 稽核程序: 遵循專業的稽核程序，確保稽核工作的客觀性和獨立性。
4. 文件記錄: 妥善保存所有稽核文件，包括稽核計畫、工作底稿、稽核報告及相關通訊紀錄。

三、 稽核報告的撰寫與發布:

1. 報告內容: 稽核報告應明確指出稽核發現事項，包括風險、缺失及改善建議。
2. 報告發布: 稽核報告應依據公司規定發布予相關主管，並抄送稽核委員會 (如有)。
3. 報告追蹤: 追蹤管理階層對稽核發現事項的改善行動，並定期評估其有效性。

四、 稽核品質控制:

1. 持續專業發展: 內部稽核人員應持續參與專業培訓，以提升其專業知識和技能。
2. 品質審查: 定期進行內部稽核工作的品質審查，以確保稽核工作的品質和一致性。
3. 獨立性: 內部稽核部門應保持獨立性，不受任何部門或個人的干預。

五、 文件管理:

  • 所有與內部稽核相關的文件應妥善保存，並依據公司檔案管理規定進行管理。

六、 程序修訂:

  • 本程序應定期評估及修訂，以確保其符合公司實際需求及最佳實務。

附則: 本程序如有任何與其他公司政策或法規衝突之處，應以較嚴格者為準。
