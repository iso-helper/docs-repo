第一章 總則

第一條 目的

本辦法旨在規範本公司（以下簡稱「公司」）與承攬商之合作關係，建立完善的承攬商管理制度，以確保工程品質、安全及效率，並維護公司權益。

第二條 適用範圍

本辦法適用於公司所有委託承攬之工程、服務或其他作業。

第三條 定義

本辦法中下列名詞之定義如下：

1. 承攬商: 指與公司簽訂契約，承攬公司工程、服務或其他作業之單位或個人。
2. 契約: 指公司與承攬商簽訂之所有相關文件，包括但不限於承攬合約、技術規範、圖說等。
3. 工程師: 指公司指派負責監督承攬商工作之人員。
4. 主管機關: 指相關主管機關或政府單位。

第二章 承攬商遴選

第四條 遴選原則

公司遴選承攬商應遵循公平、公正、公開及透明之原則，並考量承攬商之：

1. 技術能力與經驗
2. 財務狀況與信譽
3. 履約能力與績效
4. 安全管理能力
5. 合法性及社會責任

第五條 遴選程序

1. 發布招標公告或邀請投標：依據工程或服務之性質，選擇公開招標或限制性招標方式。
2. 資格審查：審查承攬商之資格文件，確認其符合資格條件。
3. 技術評估：評估承攬商之技術方案、經驗及能力。
4. 價格評估：評估承攬商之報價是否合理。
5. 評選結果公布：公布評選結果，並通知中標及未中標承攬商。
6. 簽訂契約：與中標承攬商簽訂契約。

第三章 契約管理

第六條 契約內容

契約應明確載明下列事項：

1. 工程或服務內容及規格
2. 工期及付款方式
3. 責任及權利
4. 違約責任及罰則
5. 安全衛生規範
6. 保固期限及保固責任
7. 爭議解決機制

第七條 合約變更

契約變更應以書面為之，並經雙方同意後始生效力。

第四章 作業管理

第八條 安全管理

承攬商應遵守公司及主管機關相關安全衛生規定，並應執行必要的安全措施，確保工作場所安全。

第九條 品質管理

承攬商應依契約規定及公司之品質標準執行作業，並應定期進行品質檢驗。

第十條 進度管理

承攬商應依契約規定及工程進度表執行作業，並定期向公司報告進度。

第五章 評估與考核

第十一条 績效評估

公司將定期對承攬商之績效進行評估，評估項目包括：

1. 工程品質
2. 工作進度
3. 安全管理
4. 合作態度

第十二條 考核結果

考核結果將作為未來遴選承攬商之重要參考依據。

第六章 違約責任

第十三條 違約處理

承攬商如有違反契約規定者，公司得依契約規定追究其違約責任，包括但不限於：

1. 警告
2. 罰款
3. 終止契約

第七章 附則

第十四條 爭議處理

契約履行過程中發生爭議時，應優先協商解決；協商不成者，得依契約規定或相關法律規定解決。

第十五條 修正

本辦法經公司核准後實施，並得隨時修正。

第十六條 附則

本辦法未規定事項，悉依相關法令規定辦理。
