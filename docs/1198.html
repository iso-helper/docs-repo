目的: 建立一套完善的有害物質變更管理程序，以確保產品符合相關法規要求，並有效控制有害物質的使用，降低環境風險及保護員工健康。

適用範圍: 本程序適用於所有涉及有害物質之產品設計、採購、生產、銷售及廢棄處理的相關部門及人員。

定義:

  • 有害物質: 指根據相關法規 (例如 RoHS、REACH、CPSIA 等) 規定限制或禁止使用的物質，例如鉛、汞、鎘、六價鉻、多溴聯苯(PBBs)、多溴二苯醚(PBDEs) 等。
  • 產品: 指公司生產、銷售或使用的所有產品及零組件。
  • 供應商: 指提供產品、材料或服務的廠商。

程序:

一、 變更申請:

  1. 1. 提出申請: 任何部門或人員發現需要變更產品中使用的物質 (包含新增、替換或移除有害物質)，必須填寫「有害物質變更申請表」，並提交相關資料，例如：
    • 原材料/組件規格書
    • 新材料/組件規格書 (包含有害物質成分分析報告)
    • 替代物質的安全性評估報告
    • 預期影響評估
  2. 2. 審核: 相關部門 (例如：研發部門、採購部門、品質部門、環保部門) 根據申請表及相關資料進行審核，評估變更對產品性能、法規符合性及環境影響的影響。
  3. 3. 批准/駁回: 審核通過後，由授權人員批准變更；若審核未通過，則需說明駁回原因並提出改進方案。

二、 變更實施:

  1. 1. 通知供應商: 變更獲准後，需通知相關供應商，並要求其提供符合要求的材料或組件。
  2. 2. 生產流程更新: 更新生產流程文件，以反映物質變更。
  3. 3. 測試驗證: 對採用新材料/組件的產品進行測試驗證，確保其符合相關法規要求及性能指標。
  4. 4. 記錄管理: 所有變更申請、審核記錄、測試報告等文件需妥善保存。

三、 變更追蹤與持續改善:

  1. 1. 追蹤監控: 定期追蹤變更實施情況，確保符合預期效果。
  2. 2. 持續改善: 定期檢討本程序，並根據實際情況進行調整和優化。
  3. 3. 不符合項處理: 若發現任何不符合項，需立即採取糾正措施，並記錄處理結果。

四、 相關表格及文件:

  • 有害物質變更申請表
  • 有害物質成分分析報告
  • 替代物質安全性評估報告
  • 產品測試報告
  • … (其他相關文件)

五、 責任歸屬:

  • 研發部門: 負責新產品設計及有害物質替代方案的研發。
  • 採購部門: 負責選擇符合要求的供應商及材料。
  • 生產部門: 負責生產流程的更新及執行。
  • 品質部門: 負責產品品質管控及測試驗證。
  • 環保部門: 負責環境影響評估及法規遵循。

備註: 本程序應根據相關法規及公司實際情況進行調整和更新。
