目的: 為確保高架作業人員之安全與健康，防止發生墜落等事故，特制定本辦法。

適用範圍: 本辦法適用於所有涉及高架作業之工作場所，包含但不限於建築工地、電廠、鐵路、橋樑、通信設施等。

定義:

  • 高架作業: 指在離地面2公尺以上之場所進行之作業。
  • 高架作業人員: 指從事高架作業之人員。
  • 主管機關: 指依法主管高架作業安全之機關。
  • 作業場所負責人: 指該作業場所之管理者或負責人。
  • 安全衛生管理人員: 指負責該作業場所安全衛生管理之人員。

作業前準備:

1. 風險評估: 作業場所負責人應於作業前進行風險評估，辨識潛在危險因子，並制定相應之安全措施。
2. 作業計畫: 應依據風險評估結果，擬定詳細之作業計畫，包含作業時間、地點、人員、設備、安全措施等。
3. 人員訓練: 高架作業人員應接受主管機關認可之相關訓練，並取得合格證照。
4. 設備檢查: 所有高架作業設備（例如：鷹架、吊籃、安全帶、防墜落器材等）應定期檢查及保養，確保其功能正常。 使用前應再次檢查其完整性及安全性。
5. 安全防護措施: 應依據作業內容，設置必要的安全防護措施，例如：安全網、護欄、安全帶、防墜落裝置等。
6. 緊急應變計畫: 應制定緊急應變計畫，並定期演練，以應付突發狀況。

作業中注意事項:

1. 遵守規定: 高架作業人員應遵守作業計畫及安全規定。
2. 正確使用設備: 應正確使用各項安全設備，並定期檢查其功能。
3. 相互監督: 作業人員應相互監督，發現危險狀況應立即報告。
4. 禁止違規作業: 嚴禁酒後或疲勞作業，以及任何違反安全規定的行為。
5. 天候因素: 惡劣天候(如強風、暴雨、雷雨)下應停止高架作業。
6. 定期休息: 應適時休息，避免疲勞作業。

作業後事項:

1. 設備清理: 作業結束後，應將設備清理乾淨並妥善保管。
2. 現場整理: 應將作業現場整理乾淨，清除所有雜物及廢棄物。
3. 事故報告: 如有任何事故發生，應立即報告主管機關及作業場所負責人，並配合調查。
4. 紀錄保存: 應妥善保存作業紀錄，包含風險評估報告、作業計畫、人員訓練紀錄、設備檢查紀錄等。

罰則: 違反本辦法之規定者，將依相關法規予以處罰。

附則: 本辦法如有未盡事宜，得由主管機關另行規定。
