一、 目的

為確保施工架作業之安全，防止發生墜落、倒塌等意外事故，特制定本作業辦法。

二、 適用範圍

本辦法適用於本公司所有新建、改建、修建工程中所使用之施工架之搭設、使用、維護及拆除作業。

三、 責任

1. 工程負責人： 負責施工架作業之整體安全管理，確保作業符合法規及本辦法之規定。
2. 施工架架設負責人： 負責施工架之設計、搭設、檢查及保養，並確保符合相關規範。
3. 作業人員： 負責遵守施工架作業安全規定，並正確使用安全防護具。

四、 施工架搭設

1. 設計規劃： 施工架之設計應符合相關法規及規範，並考慮工程之實際情況，如風力、地質、載重等因素。需由專業人員進行設計並繪製圖面。
2. 材料選用： 應選用合格之材料，並定期檢查其強度及完整性。禁止使用鏽蝕、損壞之材料。
3. 搭設步驟： 應依據設計圖及相關規範進行搭設，並確保每一步驟之安全穩固。搭設過程應由熟練之作業人員進行，並嚴格遵守安全操作程序。
4. 基礎處理： 施工架基礎應穩固，並應根據地質條件採取適當的基礎處理措施，例如打樁、墊塊等。
5. 垂直度及水平度： 搭設完成後，應檢查施工架之垂直度及水平度，確保其穩定性。
6. 防護措施： 應設置必要的防護措施，例如安全網、護欄、防墜落裝置等，以防止人員墜落。
7. 載重限制： 應明確標示施工架之最大載重，並嚴格遵守。不得超載使用。

五、 施工架使用

1. 人員進入： 進入施工架作業區之人員應穿戴安全帽、安全帶及其他必要之安全防護具。
2. 物料搬運： 物料搬運應小心謹慎，避免碰撞施工架，造成損壞或倒塌。
3. 定期檢查： 應定期檢查施工架之狀況，發現任何損壞或異常情況應立即停止使用並進行修復或更換。
4. 不良天候： 遇有強風、暴雨、地震等不良天候情況，應立即停止施工架作業，並採取必要的防護措施。

六、 施工架維護

1. 定期保養： 應定期對施工架進行保養，檢查其螺栓、扣件等是否鬆動，並及時進行維修。
2. 損壞修復： 發現損壞應立即修復，修復後應再次檢查其安全性。
3. 記錄管理： 應建立施工架之維護保養記錄，以便追蹤其使用狀況。

七、 施工架拆除

1. 拆除計畫： 應制定詳細的拆除計畫，並明確拆除步驟及安全措施。
2. 分段拆除： 應分段拆除，確保安全穩固。
3. 人員配合： 拆除作業應有足夠的人員配合，並嚴格遵守安全操作程序。
4. 殘材清理： 拆除完成後，應清理現場殘材，並確保現場清潔安全。

八、 罰則

  • 違反本辦法之規定，將依公司相關規定予以處罰。

九、 附則

  • 本辦法自公布之日起施行，如有未盡事宜，另行補充規定。
