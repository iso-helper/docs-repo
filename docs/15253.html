目的: 建立一套有效率且一致的安全衛生諮詢與溝通程序，以確保所有員工了解並遵循安全衛生政策與程序，預防職災發生，營造安全健康的職場環境。

適用範圍: 本程序適用於所有員工、承攬商及訪客。

程序:

一、 安全衛生諮詢:

1. 定期諮詢: 公司應定期舉行安全衛生諮詢會議，頻率至少為每月一次，由主管、安全衛生人員及員工代表共同參與。會議內容應包含：

  • 檢討過去一個月的安全衛生績效，包含事故、險近失事件及改善事項。
  • 針對潛在危險源進行風險評估與管控措施討論。
  • 宣導最新的安全衛生法規及公司政策。
  • 收集員工對安全衛生事項的意見及建議，並記錄在案。
  • 追蹤上一次會議決議事項的執行情況。

2. 不定期諮詢: 除定期諮詢外，如有任何安全衛生相關事件發生 (例如：事故、險近失事件、新設備引進、新工作程序導入等)，應立即召開不定期諮詢會議，進行討論及改善。
3. 個別諮詢: 主管應定期與員工進行個別諮詢，了解員工在安全衛生方面遇到的問題，並提供必要的協助與指導。
4. 員工回報機制: 公司應建立完善的員工回報機制，鼓勵員工主動回報任何安全衛生相關問題或事件，並確保回報管道暢通、匿名且安全。 此機制可包含：

  • 安全衛生建議箱
  • 線上回報系統
  • 定期舉辦安全衛生意見調查

二、 安全衛生溝通:

1. 宣導管道: 公司應利用多種管道宣導安全衛生相關資訊，例如：

  • 張貼海報、告示
  • 舉辦教育訓練
  • 發送電子郵件、簡訊
  • 內部網站或看板
  • 安全衛生手冊

2. 教育訓練: 公司應定期提供員工必要的安全衛生教育訓練，內容應包含：

  • 安全衛生相關法規
  • 公司安全衛生政策及程序
  • 危害辨識與風險評估
  • 緊急應變措施
  • 個人防護具的使用

3. 語言及文化考量: 針對外籍勞工或不同語言背景的員工，應提供翻譯或以其他適當方式傳達安全衛生資訊，確保所有員工都能充分理解。
4. 溝通技巧: 所有主管及員工都應學習有效的溝通技巧，以確保安全衛生訊息能清晰、準確地傳達。

三、 文件管理:

1. 所有安全衛生諮詢與溝通的記錄應妥善保存，並依規定期限保存。
2. 相關文件應定期檢討與更新，以確保其有效性。

四、 責任歸屬:

1. 最高主管: 負責推動安全衛生政策的實施，並提供必要的資源。
2. 安全衛生管理人員: 負責規劃、執行及監督安全衛生相關工作。
3. 各級主管: 負責監督其所轄員工遵守安全衛生規定，並提供必要的指導與協助。
4. 所有員工: 負責遵守安全衛生規定，並主動回報任何安全衛生相關問題或事件。

五、 程序修訂:

  • 本程序將定期檢討並根據需要進行修訂。
