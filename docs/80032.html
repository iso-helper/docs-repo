GMP (優良製造規範) 內部稽核程序涵蓋計畫、執行和報告三個主要階段，每個階段都包含多個步驟以確保稽核的有效性和客觀性。以下是詳細說明：

一、計畫階段：

1. 稽核範圍界定: 明確本次稽核的目標、範圍和時間表。這包括要稽核的部門、流程、系統以及相關的法規和標準 (例如，cGMP、ISO 9001, ISO 14001 等)。 範圍需要根據風險評估來決定，優先稽核高風險區域。

2. 稽核團隊組成: 選擇具備相關專業知識和經驗的稽核員。 避免稽核員與被稽核部門有直接利害關係，以確保客觀性。團隊成員應包括領導、記錄員和必要時專家。

3. 稽核計畫制定: 制定詳細的稽核計畫，包括：

稽核時間表: 明確稽核開始和結束日期、時間和地點。
 稽核清單 (Checklist): 基於GMP要求和相關法規，制定詳細的稽核清單，涵蓋所有需要檢查的項目。清單應包含具體的稽核點和評估標準。
資源準備: 確保所有必要的資源，例如稽核清單、文件、審查材料、設備等。
 溝通計畫: 制定如何與被稽核部門溝通的計畫，以確保順利進行稽核。

4. 風險評估: 在計畫階段應進行風險評估，識別可能影響GMP合規性的風險，並將其納入稽核計畫中。高風險區域應獲得更多的稽核時間和關注。

二、執行階段：

1. 稽核開始會議: 在稽核開始前，與被稽核部門召開會議，介紹稽核目的、範圍、時間表以及稽核程序。

2. 文件審查: 審查相關的文件，例如標準作業程序 (SOP)、記錄、報告等，以驗證其完整性、準確性和一致性。

3. 現場觀察: 對生產流程、設備、設施進行現場觀察，驗證其符合GMP要求。

4. 訪談: 與相關人員進行訪談，以了解其工作程序、遇到的問題和建議。

5. 取樣 (如有需要): 根據需要，進行產品或材料取樣，進行實驗室測試。

6. 證據收集: 收集所有相關的證據，包括文件、觀察記錄、訪談記錄、照片等。

7. 稽核中發現問題的處理: 及時記錄任何發現的不符合事項，並與被稽核部門討論可能的糾正和預防措施。

8. 稽核結束會議: 在稽核結束後，與被稽核部門召開會議，討論稽核結果，並確認任何需要糾正的發現。

三、報告階段：

1. 稽核報告編寫: 編寫一份詳細的稽核報告，包括以下內容：

  • 稽核日期和時間:
  • 稽核範圍:
  • 稽核團隊成員:
  • 稽核結果: 包括符合事項和不符合事項 (非符合項)，並附上相關證據。
  • 觀察到的問題: 描述觀察到的問題，並進行詳細說明。
  • 建議的糾正和預防措施 (CAPA): 提出糾正和預防措施，以防止類似問題再次發生。
  • 稽核員簽名和日期:

2. 報告分發: 將稽核報告分發給相關部門和管理層。

3. 管理評估和回應: 管理層需審閱稽核報告，評估報告中提出的不符合事項和建議，並制定糾正和預防措施。

4. 糾正和預防措施的跟蹤: 追蹤糾正和預防措施的實施情況，確保其有效性。

5. 稽核報告的保存: 根據公司文件管理系統，妥善保存稽核報告。

持續改進: GMP內部稽核程序應定期審查和改進，以確保其有效性和適應性。稽核結果應被用於持續改進公司的GMP體系。
