目的:

  • 本程序文件旨在確立內部品質稽核的流程，以評估組織的品質管理系統 (QMS) 是否符合既定的標準、法規要求及組織自身的目標。透過持續改進，確保產品/服務品質及符合客戶需求。

範圍:

  • 本程序適用於組織所有部門及相關活動，涵蓋[列出適用部門及活動，例如：研發、生產、銷售、客服等]。

責任:

  • 稽核主管: 負責規劃、執行及監督內部稽核活動，並編寫稽核報告。
  • 稽核員: 負責根據稽核計劃執行稽核工作，收集證據，並撰寫稽核發現報告。
  • 被稽核部門主管: 負責提供稽核所需資料及配合稽核工作，並對稽核發現採取糾正及預防措施。
  • 管理階層: 負責審查稽核報告，並批准糾正及預防措施的實施。

程序:

1. 稽核計劃:

  • 每年制定稽核計劃，明確稽核範圍、目標、時間表、稽核員及被稽核部門。
  • 計劃應考慮風險評估結果及組織的變更情況。
  • 稽核計劃應經管理階層批准。

2. 稽核執行:

  • 稽核員應具備相關的知識、技能及經驗。
  • 稽核活動應根據預先確定的稽核清單或稽核方案進行。
  • 稽核應包含文件審查、現場觀察及人員訪談等方法。
  • 稽核員應客觀地記錄稽核發現，並收集相關證據。

3. 稽核報告:

  • 稽核員應在稽核完成後，編寫稽核報告，包括稽核範圍、日期、方法、發現事項、證據及建議的糾正及預防措施。
  • 稽核報告應經稽核主管審核並簽名。

4. 糾正及預防措施:

  • 被稽核部門主管應對稽核發現採取必要的糾正及預防措施。
  • 糾正措施應有效解決發現的問題，並防止再次發生。
  • 預防措施應針對潛在問題，防止其發生。
  • 糾正及預防措施的有效性應進行驗證。

5. 管理審查:

  • 稽核報告應提交管理階層審查。
  • 管理階層應審查稽核結果，評估QMS的有效性，並決定是否需要採取進一步的行動。

6. 文件管理:

  • 所有與內部品質稽核相關的文件，例如稽核計劃、稽核報告、糾正及預防措施記錄等，應妥善保存及管理。

7. 記錄:

  • 稽核計劃
  • 稽核報告
  • 糾正及預防措施記錄
  • 稽核員培訓記錄
