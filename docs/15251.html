目的： 建立一套系統性的安全衛生管理審查程序，定期評估安全衛生管理系統的有效性，持續改善工作場所的安全衛生狀況，預防職業災害發生。

範圍： 本程序適用於所有[公司名稱/部門名稱]之場所及相關人員。

程序：

一、 審查規劃:

1. 審查頻率： 每年至少進行一次全面性安全衛生管理系統審查，必要時可進行較高頻率的局部審查。
2. 審查團隊組成： 成立審查團隊，成員應包含：

  • 管理階層代表 (至少一位)
  • 安全衛生管理人員
  • 現場作業人員代表
  • 其他相關部門人員 (視需要)

3. 審查範圍： 涵蓋安全衛生管理系統所有面向，例如：

  • 安全衛生政策及目標
  • 法規遵循情況
  • 風險評估與管制措施
  • 緊急應變計畫
  • 教育訓練成效
  • 事故調查與分析
  • 安全衛生績效指標
  • 設備與設施維護
  • 個人防護具的使用
  • 承攬商安全管理

4. 審查文件準備： 審查前，相關部門應準備必要的資料，例如：

  • 安全衛生管理手冊
  • 風險評估報告
  • 事故報告
  • 教育訓練紀錄
  • 稽查紀錄
  • 績效指標數據

二、 審查執行:

1. 開幕會議： 審查團隊成員參與開幕會議，確認審查範圍、時間表及程序。
2. 文件審查： 審查團隊依據已準備好的文件，評估安全衛生管理系統的有效性及符合性。
3. 現場查核： 審查團隊進行現場查核，觀察實際作業情況，確認風險管制措施的執行情況。
4. 訪談： 審查團隊可訪談相關人員，了解其對安全衛生管理系統的意見及建議。
5. 記錄： 所有審查發現事項、建議事項及決議事項應詳細記錄於審查報告中。

三、 審查結果與改善行動:

1. 閉幕會議： 審查團隊於閉幕會議中，討論審查結果，確認需改善事項。
2. 改善行動計畫： 針對審查中發現的缺失，制定改善行動計畫，包含：

  • 改善措施
  • 負責單位
  • 完成期限
  • 預期效果

3. 追蹤與驗證： 追蹤改善行動的執行進度，並定期驗證改善效果。
4. 管理階層審查： 將審查報告提交給管理階層審查，並取得其認可。

四、 文件管理：

  • 所有與安全衛生管理審查相關的文件，應妥善保存及管理。

附註： 本程序應定期檢討與更新，以確保其有效性。
