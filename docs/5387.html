目的： 建立一套系統化的程序，以設定、執行、檢討及改善環境目標與標的，確保本組織持續改善環境績效，符合法規要求及利害關係人期望。

適用範圍： 本程序適用於本組織所有部門及人員，涵蓋所有環境面向之目標及標的設定與管理。

程序步驟：

一、 環境目標設定:

1. 評估現況： 評估組織現有的環境績效，包括能源使用、廢棄物產生、用水量、空氣排放等，並識別關鍵環境議題。此評估應參考法規要求、利害關係人期望、組織策略及前一年度績效檢討結果。 可運用環境影響評估、物質平衡分析等工具。
2. 目標設定： 基於現況評估結果，設定具體、可衡量、可達成、相關且有時限 (SMART) 的環境目標。目標應與組織整體策略相符，並涵蓋組織的重大環境影響。
3. 優先順序排序： 依據重要性及可行性，對所設定的環境目標進行優先順序排序。
4. 資源配置： 為達成環境目標，分配必要的資源，包括人力、財力及物力。
5. 目標宣佈： 將設定的環境目標及相關資訊，宣佈給所有相關人員，以確保充分理解與配合。

二、 環境標的設定及執行:

1. 標的分解： 將環境目標分解成可操作的環境標的，明確分配責任單位及完成期限。
2. 執行方案制定： 針對每個環境標的，制定詳細的執行方案，包括具體措施、執行時程、負責人員及所需資源。
3. 執行監控： 定期監控環境標的執行進度，並記錄相關數據。 可運用稽核、數據追蹤等方法。
4. 資源調整： 根據監控結果，適時調整資源配置及執行方案，以確保達成環境標的。

三、 環境目標及標的檢討：

1. 定期檢討： 定期(例如每年)檢討環境目標及標的達成情況，分析達成狀況及原因。
2. 績效評估： 評估環境目標及標的的達成率，並分析其對環境績效的貢獻。
3. 缺失分析： 針對未達成的標的，分析其原因，並提出改善措施。
4. 持續改善： 根據檢討結果，持續改善環境目標及標的設定及執行程序，以提升環境績效。
5. 文件記錄： 完整記錄所有環境目標、標的設定、執行、監控及檢討過程，並妥善保存相關文件。

四、 文件及記錄：

  • 環境目標設定紀錄表
  • 環境標的執行方案
  • 環境監控數據記錄表
  • 環境目標及標的檢討報告
  • 相關會議紀錄

五、 程序修訂：

本程序應定期檢討並修訂，以確保其有效性和適用性。修訂應經由相關部門主管審核並批准。
