1. 目的

本程序文件旨在規定執行ISO 55001資產管理體系管理評審的程序，確保資產管理體系持續符合性、充分性及有效性。

2. 範圍

本程序適用於[公司名稱]所有部門和人員，涵蓋所有與資產管理相關的活動。

3. 參考文件

ISO 55001:2014 資產管理體系
 [公司內部相關文件，例如資產管理政策、程序等]

4. 定義

管理評審: 系統性的審查資產管理體系的有效性、充分性和持續符合性。
 管理代表: 負責資產管理體系的管理代表。
審核小組: 負責進行管理評審的團隊。

5. 程序

5.1 規劃管理評審:

頻率: 管理評審至少每年進行一次，或根據組織的風險狀況和資產重要性調整頻率。
時間安排: 由管理代表負責規劃並安排管理評審的時間和地點。
 資料收集: 在管理評審前，應收集以下資料：
資產管理體系績效指標數據 (KPI)
 內部審核結果報告
管理評審記錄
 糾正措施和預防措施的有效性評估
法律法規及其他要求的變化
 利益相關者反饋
資產管理系統的變更記錄
 資源的充足性評估
提升資產管理體系有效性的機會
 審核小組組成: 審核小組成員應包括不同部門和層級的人員，以確保客觀性和全面性。 審核小組至少應包含一位具有資產管理專業知識的成員。

5.2 進行管理評審:

會議召開: 管理代表主持管理評審會議。
 審查資料: 審核小組審查收集的資料，評估資產管理體系的符合性、充分性和有效性。
討論與分析: 審核小組討論資料，分析體系的優勢、不足、機會和威脅 (SWOT 分析)。
 決策制定: 管理評審會議需確定必要的改進措施，並指派責任人及完成期限。

5.3 管理評審報告:

記錄編寫: 管理評審會議記錄應包括：日期、時間、地點、參與人員、議程、討論內容、決策事項、指派責任人及完成期限。
 報告發布: 管理評審報告應由管理代表簽署並發布給相關人員。
行動追蹤: 管理代表需追蹤管理評審中提出的改進措施的執行情況。

5.4 糾正措施:

  • 執行: 根據管理評審結果，需執行必要的糾正措施和預防措施。
  • 驗證: 需驗證糾正措施和預防措施的有效性。

6. 記錄控制

所有管理評審相關記錄應按照公司文件控制程序進行管理和保存。
