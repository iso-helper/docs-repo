目的: 建立一套完善的安全衛生績效量測與監督管理程序，以有效預防職業災害，保障員工身心健康，提升企業安全衛生管理水平。

適用範圍: 本程序適用於所有員工及承攬商，涵蓋公司所有營運場所及相關作業活動。

程序:

一、績效指標設定:

1. 風險評估: 定期進行風險評估，識別潛在危害，並依據風險等級設定相應的安全衛生目標及指標。 評估應涵蓋所有作業流程、設備、環境及物質。
2. 指標選定: 選擇具體、可衡量、可達成、相關性和有時限 (SMART) 的績效指標，例如：

  • 事故率: 包含輕微、重大及致死事故率。
  • 損失工時率: 衡量因事故造成的工時損失。
  • 近失事故率: 紀錄未造成事故但有潛在風險的事件。
  • 安全教育訓練參與率: 追蹤員工參與安全教育訓練的比例。
  • 個人防護具使用率: 監控員工使用個人防護具的情況。
  • 環安衛稽查發現缺失率: 追蹤稽查中發現的缺失數量及嚴重程度。
  • 緊急應變演練參與率及及格率: 評估緊急應變能力。
  • 職工健康檢查覆蓋率及異常率: 監控員工健康狀況。
  • 環安衛提案採納率: 鼓勵員工參與環安衛改善。

3. 目標設定: 根據風險評估結果及產業基準，設定具體、可衡量且具有挑戰性的安全衛生目標。
4. 指標權重設定: 根據指標的重要性，設定不同權重，以更精確地反映整體安全衛生績效。

二、資料收集與分析:

1. 資料來源: 收集事故報告、近失事故報告、稽查記錄、健康檢查報告、教育訓練記錄、個人防護具使用記錄等相關資料。
2. 資料記錄: 建立完善的資料記錄系統，確保資料的準確性和完整性。 可利用電子表格、資料庫或專用軟體進行管理。
3. 資料分析: 定期分析收集到的資料，找出安全衛生績效的趨勢和問題，並找出潛在的風險因素。

三、監督與管理:

1. 定期檢討: 定期(例如每月、每季或每年)檢討安全衛生績效，分析指標達成情況，並找出需要改進的地方。
2. 績效回饋: 將績效結果及時回饋給相關部門和員工，並採取相應的改進措施。
3. 獎勵與懲罰機制: 建立合理的獎勵與懲罰機制，鼓勵員工積極參與安全衛生管理，並對違規行為進行處罰。
4. 持續改善: 基於資料分析和定期檢討結果，持續改進安全衛生管理系統，以降低事故發生率，提升安全衛生績效。
5. 管理階層責任: 高階管理層需承擔安全衛生管理的最終責任，並提供必要的資源和支持。

四、文件管理:

1. 所有相關文件，包括風險評估報告、績效指標設定文件、資料記錄、檢討報告等，應妥善保存和管理。

五、程序修訂:

  • 本程序應定期檢討和修訂，以確保其有效性和適時性。
