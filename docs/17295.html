一、 目的

為確保堆高機操作安全，防止人身傷亡及財物損失，特制定本作業辦法。

二、 適用範圍

本辦法適用於所有使用堆高機之人員及相關作業區域。

三、 作業人員資格

1. 必須取得相關主管機關核發之堆高機操作證照。
2. 經過公司安全教育訓練並考核合格。
3. 熟悉堆高機的操作規範及維護保養知識。
4. 身體健康，無妨礙操作堆高機之疾病。

四、 堆高機使用規定

1. 操作前檢查: 每次操作前，應仔細檢查堆高機之各項功能，包含：

  • 輪胎狀況 (胎壓、磨損)
  • 剎車系統
  • 油液 (液壓油、引擎油)
  • 警示燈及喇叭
  • 起升裝置
  • 叉齒
  • 安全裝置 (安全帶、警示燈等)
  • 發現異常應立即停止使用並通報主管維修。

2. 操作過程:

  • 嚴禁酒後或服用藥物影響操作能力時操作堆高機。
  • 操作速度應依現場情況調整，避免過快造成危險。
  • 轉彎時應減速慢行，並注意周圍環境。
  • 叉取貨物時，應確認貨物重量及重心，避免超載或傾覆。
  • 叉取貨物時，應將叉齒完全插入貨物下方，並保持貨物平衡。
  • 搬運貨物時，應注意上方及周圍是否有障礙物，避免碰撞。
  • 嚴禁在堆高機上載人或攜帶非作業相關物品。
  • 嚴禁在斜坡或不平坦的地面高速行駛。
  • 上下坡時，應低速行駛，並注意車身平衡。
  • 停放堆高機時，應將叉齒放下，並拉緊剎車，熄火關閉電源。
  • 工作區域內，應設置明顯的警示標誌及安全措施。

3. 貨物搬運:

  • 搬運貨物前，應確認貨物重量、尺寸及堆疊方式是否安全。
  • 搬運貨物時，應注意貨物穩固性，避免貨物滑落或傾倒。
  • 搬運易碎或危險物品時，應採取必要的保護措施。

五、 維護保養

1. 定期進行堆高機的保養及維護，並做好記錄。
2. 發現堆高機有故障現象，應立即停止使用，並通報主管維修。
3. 嚴禁擅自拆卸或修理堆高機。

六、 安全管理

1. 公司應定期對堆高機操作人員進行安全教育訓練。
2. 公司應建立堆高機操作及維護保養的相關規範及記錄。
3. 應設置完善的安全管理制度，並定期檢查執行情況。

七、 違規處理

  • 違反本辦法之規定者，將依公司相關規定予以處罰。

八、 附則

  • 本辦法如有未盡事宜，得隨時修正補充之。
